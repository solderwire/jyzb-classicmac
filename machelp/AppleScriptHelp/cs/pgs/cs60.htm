编写和编辑命令语言

    无论您是从现有的命令语言、录制操作开始，或是从编写自己的命令语言开始，都需要使用“命令语言编辑程序”来编辑命令语言。 使用“命令语言编辑程序”编写和编辑命令语言的方法与使用其他文本编辑器的方法是一样的。

断行

    使用连续字符 (¬) ，在命令语言中给较长的行断行。 运行命令语言时，AppleScript 将该行视作一行。 下面是一个例子：

    make new folder at desktop with properties ¬
    {name:"My Folder", label index:0, selected:true,     ¬
    comment:"A folder for my new files!"}

    要在一行中插入连续字符：
    • 按下 Option ，然后按 Return 键。

使行缩排

    在编写命令语言时，“命令语言编辑程序”可以缩排复合语句的行以显示出它的结构。

    要缩排复合语句中的行：
    • 在每一行的结尾按 Return 键。

格式化命令语言

    在对一段命令语言检查语法时，“命令语言编辑程序”会格式化命令语言中不同类型的术语。 您可以改变格式化命令语言的方法。

    要改变命令语言的格式：
    1 打开“编辑”菜单，选取“AppleScript
    格式化…”。
    2 点按列表中的一个项目，然后使用“字体”和“字形”菜单来选择如何格式化。

    所做的改变会应用到您打开的所有命令语言。 不能只格式化命令语言的一部分。 您可以格式化“描述”区域中的文本。
