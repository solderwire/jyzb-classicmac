使用词典

    要知道哪些命令和对象可以用来控制程序，可查看该程序的词典。

    要打开一个词典：
    1 在“命令语言编辑程序”中，打开“文件”菜单，再选取“打开词典…”。
    2 在对话框中找到该程序并打开它。
    3 要查看关于一个术语或术语套的细节，请点按以选定它。

    请为我在“命令语言编辑程序”中打开词典。

    如果不能打开某个程序的词典：
    该程序不支持命令语言，而且没有词典。

阅读词典

    在词典中，术语被组织为称为“术语套”的组别。命令以普通文本显示。 对象以斜体显示。

    要查看术语或术语套的定义：
    • 点按以选定它。
    定义出现在窗口的右侧。

    该定义包含对术语的简短说明（加底线的文本）和术语的语法。 语法是在命令语言中必须采用书写术语的方法。 语法可能包括不同的部分。 每个部分具有一段描述，以注释的形式出现在语法后面。

    命令定义：
    在上面的例子中，Quit 命令有一个简单的语法。 大部分命令的语法均包含使用该命令所需的其他信息。 例如，Count 命令的语法为：

    count 引用 -- 元素要计入的对象
    each 每种类型 -- 要计入的元素类型
    Result: 整数 -- 元素数量

    粗体字是在语句中使用的术语，写法必须与它们在词典中出现的样子完全相同。 普通文本是您提供的信息。 例如，“reference”说明要计数的对象的名称。 破折号右侧的文本是一段描述。

    括号中的术语是可选的：

    computer type class -- the attribute to test
    [has integer] -- test specific bits of response
    [Result: integer] -- the result of the query

    在这个例子中，对象有复数形式，这样您可以在命令语言中引用多个应用程序。 一些对象没有复数形式。

    对象可以包含元素的列表，即对象中所包含的对象。 元素名称以粗体显示，而元素后面的普通文本描述如何在命令语言中引用该元素。

    对象具有一些属性或特性，供您在命令语言中使用。 属性的语法以粗体字显示。 属性后面的普通文本说明属性包含的数值类型。 它可能是字符串、数字、布尔值（真或假）或引用。 在某些情况下，属性可能具有一个固定的值，称为“常量”或常量的列表。

    一些属性在语法后包含“[r/o]”。 这表示该属性为“只读”。在命令语言中，您可以获取只读属性的值，但不能给它赋值。
