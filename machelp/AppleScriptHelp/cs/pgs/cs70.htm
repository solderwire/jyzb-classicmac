存储命令语言

    要存储命令语言：
    1 打开“文件”菜单并选取“存储”或“存储为”。
    2 使用“格式”菜单选择如何存储命令语言：
    • 存储为已编译的命令语言，这样则只能在“命令语言编辑程序”中运行。
    • 存储为传统的应用程序
    • 存储为 Mac OS X 应用程序
    • 存储为文本文件，这样则可以在其他应用程序中打开
    3 如果要将命令语言存储为应用程序，请选择应用程序的选项。
    4 为命令语言命名，选择存储的位置并点按“存储”。

    您也可以将命令语言存储为只运行的命令语言。如果将命令语言存储为只运行的命令语言，则其他用户不能在“命令语言编辑程序”中打开。

    要将命令语言存储为只运行的命令语言：
    • 打开“文件”菜单并选取“存储为只运行”。

将命令语言存储为应用程序

    在将命令语言存储为应用程序（有时称为 applet）时，可以有几种存储方式选择。

    选择一种应用程序类型：
    您可以将命令语言存储为传统的应用程序或 Mac OS X 应用程序：
    • 存储为传统的应用程序，您的命令语言可以在支持 AppleScript 的任何系统中运行。
    • 存储为 Mac OS X 应用程序，这样您的命令语言只可以在装有 CarbonLib（包括 Mac OS X）的计算机上运行。

    存储拖放执行应用程序：
    要运行存储为应用程序的命令语言，可以连按它。如果命令语言中有 On Open 处理程序，则您可以通过将文件或文件夹拖移到它上面来运行该应用程序。这种应用程序有时称为“droplet”（拖放执行程序）。

    选择选项：
    在将命令语言存储为应用程序时，可以选择以下任一个选项：
    • 保持打开
    选择该选项可使命令语言运行后仍旧保持打开。如果要从其他命令语言发送命令给打开的命令语言，则可以选择该选项。
    • 永不显示启动屏幕
    如果您不想显示启动屏幕（显示命令语言“描述”区域的内容），请选择该选项。
