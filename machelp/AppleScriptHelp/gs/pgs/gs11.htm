支持命令语言的程序

    一个程序要支持命令语言，必须具有一套定义的 AppleScript 术语，让您在为该程序编写命令语言中使用。

    Finder（或“桌面”）有许多 AppleScript 术语，让您使用命令语言来管理文件和文件夹，并设置桌面上的预置。

    您还可以使用命令语言自动化 Mac OS 中包含的以下网络和 Internet 程序的任务：
    • Sherlock
    • 用户与组别
    • 文件共享
    • 远程访问
    • Web 共享

    下面列出了一些在 Mac OS 中支持命令语言的其他程序：
    • 应用程序切换器
    • “位置管理程序”控制板
    • “外观”控制板
    • “苹果菜单选项”控制板
    • “文件交换程序”控制板
    • “键盘”控制板
    • 苹果系统概述
    • ColorSync
    • 桌面打印

    另外，“附加命令语言”文件夹（在“系统文件夹”中）包含一些程序，提供可在命令语言中使用的术语。 附加命令语言可扩充 AppleScript 语言并且随时可用，所以不用将附加命令语言定义的命令指向指定的应用程序。

    Macintosh 计算机上可以使用的许多其他程序都支持命令语言。 要确定一个程序是否支持命令语言，看是否可以在“命令语言编辑程序”中打开它的词典。
