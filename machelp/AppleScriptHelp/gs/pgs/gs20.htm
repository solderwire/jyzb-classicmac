新内容

    这一小节简短地描述了这个版本的 AppleScript 所包含一些新功能。

    如果要获得关于新功能的更多信息，请访问 AppleScript 网站：www.apple.com/applescript（英文）。

系统要求

    这次推出的 AppleScript 版本为 1.5.5，需要 Mac OS 9.1 或更高版本。它不能用于以前版本的 Mac OS。

Mac OS X 支持

    这一版本的 AppleScript 新增了对 Mac OS X 的支持：
    • 除所有可编写命令语言的 Mac OS 9 应用程序之外，您可以在 Tell 程序块中指定也能配合 Mac OS 9 使用的可编写命令语言的 Mac OS X 应用程序。您也可以指定 Mac OS X 桌面应用程序。
    如果您将命令语言存储为编译的命令语言或传统的 applet，您也可以在 Mac OS X 传统环境中运行它，来控制 Mac OS X 版本的应用程序或桌面程序。
    • “命令语言编辑程序”显示了可编写命令语言 Mac OS X 应用程序的词典，设计的这些应用程序也可以配合 Mac OS 9 和 Mac OS X 桌面应用程序使用。
    • “命令语言编辑程序”允许您将命令语言存储为 Mac OS X applet。您也可以在 Mac OS X 中运行命令语言，而无需启动传统环境。
    • 如果在“功能扩展”文件夹中（位于“系统文件夹”中）有 CarbonLib 功能扩展，您就可以在 Mac OS 8.1 或更高版本中运行 Mac OS X 程序。
    • 您可以使用由某些与“命令语言编辑程序”储存命令语言方法不同的 Mac OS X 应用程序创建的命令语言文件。（这些命令语言在文件的数据分支中，而不是在资源分支中）。
    这样可以更容易地使用这些命令语言文件执行某些网络操作。

使用 TCP/IP 寻址控制远程应用程序

    除使用 AppleTalk 名称来识别计算机之外，命令语言还可以使用 TCP/IP 地址控制其他计算机上的应用程序。例如：

    tell application "Finder" of machine "eppc://127.200.126.44"

    您也可以使用计算机的 DNS 名称：

    tell application "Finder" of machine "eppc://lab.mycompany.com"

    您必须知道计算机的 TCP/IP 地址或 DNS 名称。您不能通过浏览网络找到它。

Using Terms From 命令

    Using Terms From 命令可以让您为应用程序指定一个与在 Using Terms From 程序块内 Tell 命令的目标程序不同的术语。

    如果实际的应用程序不可用或您不能连接到远程应用程序，使用这个命令可以开发和编译一些命令语言。

    下面是部分样本：

    using terms from application "Finder"

      tell application remote-Finder
      ...
      end tell

    end using terms from

    AppleScript 使用 Finder 编译 Using Terms From 程序块中的语句，而不改变 Tell 程序块的目标应用程序。

    您可以在 Tell 程序块中嵌套 Using Terms From 程序块，也可以在 Using Terms From 程序块中嵌套 Tell 程序块。如果 Using Terms From 程序块在 Tell 程序块内，AppleScript 会使用 Tell 程序块指定目标中的术语。

增强的 Unicode 文本支持

    AppleScript 现在可以将应用程序中返回的 Unicode 文本转换为国际化的文本并使用已安装的多语言命令语言系统来显示结果。

    AppleScript 也可以转换包含多个命令语言系统的文本。例如，您可以将包含英语、古斯拉夫语和日语的文本转换为 Unicode 文本，然后转为国际化的文本。

    命令语言也可以比较 Unicode 文本值以测试它们是否等同。

支持大文件

    Info For（简介）命令能正确报告出大于 2 GB 的文件、文件夹和磁盘的大小信息。

Info For 命令的“忙状态”属性

    Info For 命令有一个“忙状态”属性，如果该命令指定的文件正在使用，它会返回 True 值。

Rouding（四舍五入）选项

    Round 命令（位于“标准附加”中）新增加了 as taught in school 参数，可以将以 .5 结尾的分数四舍五入到下一个最相近的整数。下面是一个例子：

    round 0.5 rounding as taught in school

    如果没有这个参数，Round 命令将四舍五入到相邻偶整数。

“钥匙串访问”支持

    如果服务器和宗卷的密码存储在“钥匙串访问”中，您可以使用 Mount Volume 命令来装上 AFP（AppleShare）服务器宗卷，而无需输入密码。这可以让您装上命令语言中不包括密码信息的网络宗卷。

文件夹操作术语更改

    Remove Actions From 和 Edit Actions Of 命令（在 Folder Actions 词典中）的“action number”和“action name”属性改为“using action number”和“using action name”。

DOS 和 UNIX 文本支持

    这个版本的 AppleScript 可以检测到 DOS 和 UNIX 系统中文本的分段符。
