控制语句

    通常 AppleScript 执行完一个语句后会执行下一个语句。 控制语句可以改变这种顺序。 您可以使用控制语句跳过语句或重复执行语句。

    控制语句可以包含其他的控制语句，所以可以根据变化的条件在命令语言中产生复杂的顺序。

    下面是可以在命令语言中使用的一些控制语句的简短描述。

Tell 语句

    Tell 语句指定命令被发送到的一个预设对象。

    tell front window of application "Finder" close

    如果不指定一个预设对象，AppleScript 将命令发送到当前的应用程序。

    与其他的控制语句一样，您可以在一个 Tell 语句中使用 Tell 语句。

If 语句

    下面例子中说明的 If 语句，让您根据一个条件控制执行哪些语句。 If 语句包含至少一个表达式，在该命令语言运行时它为真或者假。

    如果跟在 if 后的表达式为真，则执行 then 后的语句。 如果表达式为假，则不执行该语句。

    您可以在 If 语句中加入 Else 子句，在条件为假时执行 Else 后面的语句。 Else 子句必须是 If 语句的最后一部分，如下面的例子所示。

    请为我显示一个带有 Else 子句的 If 语句。

    在最后的 Else 子句前可使用 Else If 子句，以加入更多的条件和语句。 您还可以在 If 语句中嵌套 If 语句。

Try 语句

    命令可能会产生出错信息。 Try 语句让您控制在错误出现时所发生的事件。 Try 语句包含如果发生错误要执行的语句。 以下是基本的形式：

    try
    [ 语句 ]
    on error
    [ 语句 ]
    end try

    如果在执行 Try 后面的语句时发生错误，则执行在 On Error 从句后的语句。

Repeat 语句

    Repeat 语句让您重复执行一些语句。 它可以包含一个变量来控制语句执行的次数。

    Repeat 语句有几种格式：
    • Repeat （无限地）： 无限地重复执行语句。
    repeat
    [ 语句 ]
    end repeat
    在 If 语句中使用 Exit 语句，停止重复执行。
    • Repeat Number Times： 以一定的次数重复执行语句。
    repeat 整数 [ times ]
    [ 语句 ]
    end repeat
    • Repeat While： 当条件为真时重复执行语句。
    repeat while 表达式为真
    [ 语句 ]
    end repeat
    • Repeat Until： 重复执行语句直到条件是真为止。
    repeat until 表达式为真
    [ 语句 ]
    end repeat
    • Repeat With Variable： 通过一定范围的值重复语句多次。
    repeat with 循环变量 from 初值¬
    to 终值 [by 步长值]
    [ 语句 ]
    end repeat
    初值 和 终值 决定变量的范围。 步长值 决定在该范围内步长的大小。 预设值为 1。
    • Repeat With Variable In List： 根据列表定义的范围重复执行语句。
    repeat with 循环变量 in 列表
    [ 语句 ]
    end repeat
    • Exit： 停止重复执行语句。
