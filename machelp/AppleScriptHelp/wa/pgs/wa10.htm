语句

    每个命令语言是一系列的语句。 每个语句包含一个命令，也可能包括执行该命令需要的附加信息。

    语句可以很简单，例如下面这个只有一行的命令：

    tell application "Finder" to activate

    语句段通常较复杂，包含多个语句。 下面的例子就包含几个语句。

    请为我显示一个复合语句。

    在运行一段命令语言时，每个语句均按顺序执行。 然而，一些语句可以控制语句执行的顺序，例如下一个例子。

    请为我显示一个控制语句。

    这个例子包含两个控制语句：
    • Tell 语句控制什么对象接收其他的语句。 在这种情况下，Finder 应用程序就是对象。
    • If 语句控制是否执行第二个 Set 命令。 它只在变量“item_comments”为空时才执行。
