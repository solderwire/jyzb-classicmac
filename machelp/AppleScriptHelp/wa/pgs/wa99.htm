其他特征

    AppleScript 还包含几个您应该知道的特征：
    • 连续字符 (¬)： 让您断开较长的行。
    要输入连续符，键入 Option-Return。
    every file in control panels folder¬
    whose file type is "APPL"
    AppleScript 将它视作一行处理。
    • 注释： 被 AppleScript 忽略的文本。
    注释有两种形式：
    • (* 这是一块注释，应放在其他语句之间。在命令语言中，注释可以连续几行。 *)
    • quit -- 这是放在语句结尾的注释
    • 标识符： 某些东西的名称，例如一个变量、一个属性的标签或处理程序。
    标识符必须以一个字母开始。
    • 区分大小写： AppleScript 不区分大小写。
    • 缩写： AppleScript 允许您缩写一些术语。 例如，在 Tell 语句中，可以用“app”代替“application”：
    tell app "Finder" print the front window
    • 可选的字词： AppleScript 语言让您象写句子一样编写语句。 一些术语是可选的，例如前面例子中的“the”。 如果对理解命令语言有帮助，您可以使用它们，但 AppleScript 并不要求。
