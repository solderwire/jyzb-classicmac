命令

    命令是一些字词或短语，使程序执行某些任务。 下面是一段命令语言的例子，其中包含四个命令：

    请为我显示使用几个命令的命令语言。

    此例子包含以下命令：
    • Activate 使 Finder 成为现用应用程序。
    • Set 将前面窗口的名称放进一个容器，命令语言将其命名为 windowName。该容器称为“变量”，因为它可以包含任何名称。 变量是在命令语言中使用信息的简易方法。
    • Close 告诉 Finder 关闭窗口。
    • Display Dialog 显示带有指定信息的对话框。

    AppleScript 定义了五个命令： Count、Copy、Get、Set 和 Run。 其他程序和附加命令语言（在硬盘上的“系统文件夹”的“附加命令语言”文件夹中）定义了其他命令。
