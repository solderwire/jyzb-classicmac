设置控制板选项

    您可能会发现需要更改显示器的颜色层次才能使用某些软件。使用“显示器”控制板或控制条模组的“颜色层次”部分可以更改这个设置。您也可以使用 AppleScript 完成这一工作。

    “附加命令语言”文件夹（位于“系统文件夹”中）包括有一个“显示器层次”附加命令语言，定义了可以用来设置显示器颜色层次的术语。

    要查看可以使用的术语，请打开“显示器层次”附加命令语言的词典。

    请为我打开显示器层次词典。

    您会看到 System Object Suite（系统对象术语套）中定义的这些术语：
    • monitor depth: 当前显示器层次设置
    • max monitor depth: 最大显示器层次设置
    • min monitor depth: 最小显示器层次设置
    • set monitor depth: 设定显示器层次设置

    您可以使用这些术语编写简单的命令语言，将显示器层次设置为显示器允许的最大值：

    set monitorMaxDepth to max monitor depth
    set monitor depth monitorMaxDepth

    下面的样本显示一个对话框，让您选择想要的显示器层次：

    请为我显示显示器层次样本。

    注意这些语句均没有使用 Tell 语句。由于附加命令语言始终都可以使用，因此没有必要将该命令指向特定的应用程序。
