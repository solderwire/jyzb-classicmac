概括文件的内容

    如果您管理大量的文本文件，则可能会想查看文件内容的概要。 “标准附加”词典（位于“附加命令语言”文件夹）包含术语 Summarize，可用来完成该工作。

    请为我打开“标准附加”词典。

    要查看 Summarize 命令的语法：
    • 找到 String commands（字符串命令）术语套并点按 summarize。

    Summarize 命令用于概括文本。 您可以提供一个实际的文本、文本文件的替身或一个文本文件的文件说明，包括该文件所在的任何文件夹的名称。 例如，文件说明可能会如下所示：

    My Disk:Documents Folder:Text File

    您可能要使用该命令来概括任何文件，所以命令语言需要让您选择要概括的文件。 以下的例子使用了在“标准附加”词典中定义的其他两个命令：
    • Read，在 File Read/Write （文件读/写）术语套中，用以读取文件的内容
    • Choose File，在 User Interaction （用户交互）术语套中，显示一个对话框，让您选择一个文件

    请为我显示概括命令语言。

    该命令语言让您一次选择一个文件，并在“命令语言编辑程序”的“结果”窗口中显示概要。

    在没有打开“命令语言编辑程序”时，还可以查看多个文件的概要，包括文件夹中的文件。 下面的例子包括一个 On Open 处理程序，可以将其用作拖放执行的命令语言。 要使用该命令语言，将其存储为应用程序，然后在 Finder 中将文件拖曳到它上面。 该命令语言显示拖曳到它上面的任何文本文件的概要。

    请为我显示概括几个文件的拖放命令语言。
