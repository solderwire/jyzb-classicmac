文件夹操作命令语言如何工作

    在“标准附加”词典中，Folder Actions（文件夹操作）术语套包含五个命令：

    请为我打开“标准附加”词典。

    这些命令中的每一个均会将一个事件发送给“文件夹操作”功能扩展中（包含在 Mac OS 中）。 如果将包含一个或多个命令处理程序的命令语言附在一个文件夹，在相关事件发生时命令语言会运行。

    例如，如果要在打开一个文件夹时执行文件夹操作命令语言，您应在命令语言中包含 Opening Folder 命令的处理程序：

    on opening folder my_folder
    -- statements to execute here
    end opening folder

    一段命令语言可以包含五个命令中一个或多个命令的处理程序，但是每个命令只可以有一个处理程序。 例如，一段命令语言中可以有一个用于文件夹打开时的处理程序和一个用于文件夹关闭时的处理程序。

    要将文件夹操作命令语言附在文件夹上：
    1 按下 Control 键，并选择该文件夹。
    2 从出现的菜单中选择“附加文件夹操作…”。
    3 使用出现的对话框选取该命令语言。

    虽然可以将文件夹操作命令语言存储在任何地方，但您可能想要将它们保留在“文件夹操作命令语言”文件夹中。 该文件夹包含 Mac OS 附带的命令语言。 它位于“系统文件夹”的“命令语言”文件夹中。
