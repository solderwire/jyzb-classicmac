文件夹操作命令语言的例子

    这里包含由 Finder 定义的每个文件夹操作命令的例子。

打开文件夹

    此例子包含 Opening Folder 命令的处理程序。 它会获取文件夹的名称，当有项目放入该文件夹中时显示一个对话框，来告诉您。

    请为我显示 Opening Folder 命令。

关闭文件夹

    此例子包含 Closing Folder Window For 命令的处理程序。 它会获取文件夹和磁盘的名称，然后使用 text-to-speech 告诉您该文件夹已被关闭。

    请为我显示用于文件夹关闭时的命令语言。

移动文件夹

    此例子包含 Moving Folder Window For 命令的处理程序。 它会获取窗口原来的位置和窗口的新位置，并在一个对话框中显示这些信息。

    请为我显示用于文件夹移动时的命令语言。

    每个 Copy 命令都将窗口位置表示为一个具有四个数字的变量。 前两个数字定义了窗口的左上角；后两个数字定义了窗口的右下角。 这些数字以像素为单位，是从屏幕的左上角开始计算的屏幕上的点数。

给文件夹增加项目

    此例子包含 Adding Folder Items To 命令的处理程序。 它会获取所增加项目的数目和文件夹的名称，然后使用 Say 命令告诉您这些信息。

    请为我显示将项目增加到文件夹的命令语言。

从文件夹中移走项目

    此例子包含 Removing Folder Items From 命令的处理程序。 它获取文件夹的名称和移动项目的数目，然后使用 Say 命令告诉您这些信息。

    请为我显示文件夹项目被移走时的命令语言。

    此例子首先将 AppleScript 使用的文本定界符改变为冒号（:）。 “文本定界符”是一种字符，AppleScript 通常将其视为分隔字。 由于包含在参数 this_folder 中的文件路径名以冒号分开，因此命令语言可以很容易地识别最后的项目，即文件夹的名称。

    此例子还会使用 Try 命令，它可让您处理可能发生的错误情况。 由于该命令语言在 Try 命令的 On Error 部分并没有任何语句，因此它会忽略错误信息。
