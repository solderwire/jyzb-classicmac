将 USB 设备连接到计算机上

    您可以在计算机打开时连接 USB 设备。

    警告： 请不要在 USB 设备（尤其是存储设备）使用时，连接或拔下该设备，否则可能会损坏文件。

    要将 USB 设备连接到计算机上：
    1 如果需要的话，将 USB 设备接上电源。
    2 如果您的 USB 设备没有相连的 USB 电缆，请将一条 USB 电缆上的 B 类 USB 接头插到该设备相应的 USB 端口上。
    3 将 USB 电缆另一端插入计算机闲置的 USB 端口。（请查阅随计算机附带的打印文稿，找到 USB 端口的位置）。

    如果想连接的设备没有 USB 端口：
    • 您可能可以使用 USB 适配器连接非 USB 设备。

    如果计算机没有闲置的 USB 端口：
    • 添加一个 USB 集线器并将其他 USB 设备连接到集线器的端口上。
    • 不使用设备时应拔下该设备以便留出端口。

    如果 USB 设备无法工作：
    • 请参阅 USB 疑难解答部分的解决方案。
