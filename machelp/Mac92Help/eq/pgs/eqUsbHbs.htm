USB 集线器的类型

    USB 集线器是一种可让其它 USB 设备插入的设备。如果要同时将更多的设备连接到计算机上，或要增加设备和计算机之间的距离，您可以使用 USB 集线器。您可以通过将集线器插在一起组成一个大的多层 USB 设备链，以扩充可用 USB 端口的数量。您还可以使用 USB 集线器来延长连接设备的距离。

    USB 集线器共有两种类型： 自身供电型和总线供电型。

自身供电型集线器

    这种类型的集线器有一个交流适配器，负责给集线器提供外接电源。计算机是自身供电型集线器。要使用该集线器，需要将适配器连接线插到电源插座或电源插线板上。集线器会自行供电，因此您可以将大部分 USB 设备与它连接。典型的自身供电型集线器有 4 个 或 7 个 USB 端口。

    注：苹果公司建议您使用自身供电型集线器而不是总线供电型集线器，以获得最佳的性能。

    自身供电型 USB 集线器

总线供电型集线器

    这种类型的集线器从计算机或其它自身供电型集线器获得电源，因此不需要将它插入交流插座。您可以将低能耗 USB 设备（例如鼠标或游戏杆）插入总线供电型集线器。苹果 USB 键盘带有一个总线供电型集线器。

    总线供电型 USB 集线器

    重要事项： 不可以将两个总线供电型集线器接在一起，因为这样不能给每个集线器或相连的设备提供足够的电源。要将两个集线器连接起来，至少有一个必须是自身供电型集线器。因此，您不能将苹果 USB 键盘连接到另一个苹果 USB 键盘的 USB 端口上。如果想在计算机上使用两个总线供电型集线器，需把每个都接到计算机或自身供电型集线器的 USB 端口上。如果使用三个或更多的集线器相互连接，请注意使得设备链在两种类型间交替连接。
