USB 端口、接头和电缆

    USB 端口和接头有两种：Type A 和 Type B。标准 USB 电缆的一端有一个 Type A 接头，另一端有一个 Type B 接头。要获得最佳性能，请使用标准电缆。

    Type A 端口和接头是小长方形，通常用于将一个设备插到计算机上的 Type A USB 端口或集线器上。

    USB Type A 端口和接头

    Type B 端口和接头是正方形的，常用于将 USB 电缆连接到一个 USB 设备。

    USB Type B 端口和接头

    有些设备（例如苹果 USB 键盘）有一根连接的（或“固接式”）电缆，因此只有一个带 Type A 接头的可用端。

    许多设备附带一根标准 USB 电缆。但是，如果您需要购买 USB 电缆，可以向电子产品销售店、苹果授权经销商或零售商咨询。
