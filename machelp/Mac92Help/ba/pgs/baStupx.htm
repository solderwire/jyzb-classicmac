使用 Mac OS X 启动

    如果购买时计算机中已经安装了 Mac OS X，或者您自己安装了 Mac OS X，您可以在 Mac OS 9 和 Mac OS X 之间切换。

    要设置计算机使用 Mac OS X，请使用“启动磁盘”控制板。

    请为我打开“启动磁盘”控制板。

    1 点按包含操作系统文件夹的磁盘旁边的三角。
    2 选择 Mac OS X 系统作为您的启动磁盘。
    3 点按“重新启动”。

    当您设置计算机使用 Mac OS X 启动后，它将一直使用 Mac OS X 启动，直到您再次设置它使用 Mac OS 9 启动。
