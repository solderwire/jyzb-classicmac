选择网络服务

    某些应用程序允许您打开特定的 URL，或浏览网络来使用专门针对应用程序的服务或一般服务（如 AppleShare、FTP 或 HTTP）。

    要浏览服务：
    • 请展开列出的网络邻域直到您看到该服务，点按以选定它，并点按“选取”。

    要选择您查找的服务器类型：
    • 请点按“迅捷”按钮并选定服务类型。
     

    如果您查找的服务没有列出来：
    • 请检查所选的服务类型（例如 Web 服务器或 FTP 服务器）。
    • 打开 Internet 控制板，点按“高级”标签，并检查“主机”设置中的指定 LDAP（简单目录访问协议）服务器。
    • 询问您的网络管理员，确定该服务是否已在您计算机可以找到的网络段目录中注册。
    • 要了解应用程序如何找到网络服务，以及您或您的网络管理员如何使特定的服务显示在服务列表中，请参阅 Network Services Location Manager Network Administrator's Guide。在以下网站搜索这个标题：http://www.info.apple.com/manuals/manuals.taf（英文）

    提示：要查看追加到服务器列表内网络邻域的源目录类型，请按住 Control 键打开“文件”对话框。
