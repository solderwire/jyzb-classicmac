使用 Unicode 显示多语言文本

    Mac OS 9.2 支持 Unicode ，可以让您创建并打开包含特殊字符的文稿，包括多种语言的文本和数学符号。

    在这一 Mac OS 系统中，您可以查看现有的 Unicode 文稿，但如果想要输入 Unicode 字符，您必须从系统软件光盘安装软件。

    要打开 Unicode 输入：
    1 打开 Unicode 文件夹，该文件夹位于系统软件光盘的“光盘附加程序”文件夹中。
    2 将 Unicode 文件及其四个键盘布局拖移到关闭的“系统文件夹”。要打开请先阅读文件，请将 WorldText（位于系统软件光盘“光盘附加程序”文件夹内的 WorldText 文件夹中）应用程序拖移到硬盘上。
    3 将“模拟键盘” 9.0 拖移到硬盘上。
    这个新版本的“模拟键盘”可以让您输入 Unicode 字符，看到纯 Unicode 布局的键盘。您可以在同一个文件夹中保存新旧两个版本的“模拟键盘”，但要将其中一个更名。
    4 重新启动计算机。

    带有旗标图象的菜单包含有已安装的所有键盘布局。Unicode 键盘及其他键盘上的特殊字符只能供识别 Unicode 的应用程序使用（如 WorldText）。
