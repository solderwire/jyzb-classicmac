Unicode 键盘布局

    要使用 Unicode 文件夹中的键盘布局（可在系统软件光盘的“光盘附加程序”文件夹内的 Unicode 文件夹中找到），您需要启用 Unicode 命令语言并使用能显示 Unicode 字符的应用程序（如 WorldText）。

      键盘布局名   让您输入    
    Extended Roman 罗马字母系列内的多种重音字符
    Symbol Symbol 字体中的特殊字符
    Dingbats Zapf Dingbats 字体中的特殊字符
    Unicode Hex Input 您知道其四位十六进制代码的任一 Unicode 字符 您可以在 Unicode Consortium 网站：http://www.unicode.org/（英文）找到这些代码。
