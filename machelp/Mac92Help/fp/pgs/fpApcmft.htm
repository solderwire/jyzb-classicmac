关于打开 DOS 或 Windows 文件的问题

    您的计算机可以识别最常见类型的 DOS 与 Windows 文件，并使用预先指定的应用程序打开这些文件。

    如果不能通过连按文件图象打开 DOS 或 Windows 文件：
    请尝试从您要使用的程序中查看和编辑该文件。许多 Mac OS 程序可以直接打开 DOS 或 Windows 文件。

    如果 DOS 或 Windows 文件显示不正确，或者您看到奇怪的字符：
    • 您的应用程序可能要求您按照特殊的说明以其他格式打开和存储文件。请参阅随该程序附带的说明文稿。
    • 请尝试在另一个程序中打开该文稿。
    • 在 DOS 计算机上能够显示的一些字符不能在 Macintosh 计算机上正确显示。

    如果预先指定的应用程序不能正常工作：
    使用“文件交换程序”控制板的 PC Exchange 部分，更改指定的应用程序或增加新的应用程序。

    请帮我为 DOS 或 Windows 文件种类指定应用程序。

    在“文件交换程序”控制板中设定的选项会修改您在 Internet 控制板中设定的 Internet 设置。在 Internet 控制板中所作的更改也会修改您的“文件交换程序”设置。

    提示：要获得有关使用“文件交换程序”控制板的更多帮助，请打开“帮助”菜单并选择“显示旁述”。然后，指向您感兴趣的功能或菜单项目。
