提高频繁使用磁盘的程序的性能

    您可以使用 RAM 磁盘来加速经常要读写磁盘的程序的性能。RAM 磁盘可让您将计算机中的一部分内建随机存取存储器（RAM）用作高速存储磁盘。

    注意：使用 RAM 磁盘还可以节约便携式计算机上的电池能量。

建立 RAM 磁盘

    要建立 RAM 磁盘：
    • 在“存储器”控制板中打开 RAM 磁盘选项，调整 RAM 磁盘大小，然后重新启动计算机。
    请为我打开“存储器”控制板。

    在某些计算机上（例如 PowerBook），您可以在“存储器”控制板中选择“关机时存储”选项来存储 RAM 磁盘上的内容。

    警告：如果在“存储器”控制板中没有选择“关机时存储”，在关闭计算机的时候，计算机 RAM 磁盘上的内容将被抹去。请确保在关闭计算机之前已把 RAM 磁盘上的内容拷贝到了硬盘上。

抹去 RAM 磁盘上的内容

    要抹去 RAM 磁盘上的内容：
    1 连按 RAM 磁盘图象以将其打开。
    2 将您想存储的文件拷贝到硬盘或软盘上。
    3 将 RAM 磁盘上的所有东西拖到废纸篓里。
    4 打开“其它”菜单，选择“清倒废纸篓…”。

    注意：抹去 RAM 磁盘并不会删除它的图象或增加可用内存。要空出 RAM 磁盘使用的内存，您必需删除 RAM 磁盘并重新开机。

重新配置 RAM 磁盘的大小

    要重新配置 RAM 磁盘的大小：
    • 抹去 RAM 磁盘的内容，然后在“存储器”控制板中调整 RAM 磁盘大小并重新开机。

移去 RAM 磁盘

    要移去 RAM 磁盘：
    • 抹去 RAM 磁盘的内容，然后在“存储器”控制板中关闭 RAM 磁盘并重新开机。
