共享 USB 打印机

    如果您的计算机连接到 TCP/IP 网络，可以
    • 让网络上的其他 Macintosh 计算机通过连接到您计算机上的 USB 打印机来打印
    • 在连接到网络上其他 Macintosh 上的 USB 打印机打印

共享您的打印机

    要共享连接到您计算机上的 USB 打印机：
    • 打开“USB 打印机共享”控制板，检查“启动/停止”标签以确保共享已启动，然后在“我的打印机”标签中点按打印机名称旁边的“共享”注记格。
    请为我打开“USB 打印机共享”控制板。

    提示：从“我的打印机”标签拖移共享打印机到桌面上以创建一个可以发送给其他人的 Internet 位置文件。要预先选择您的打印机，其它用户拖移这个文件到他们的“USB 打印机共享”控制板上的“网络打印机”标签中。

    您的打印机必须在网络邻居中列出，以使 USB 打印机共享能工作。

    要设置您的网络邻居（或查找您的哪一个打印机已经列出）：
    • 点按下面的链接，然后在出现的对话框中输入邻居的名称。
    设置我的网络邻居。

使用共享打印机

    在使用“选配器”或“桌面打印实用程序”选定共享 USB 打印机之前，您必须预先选择这台打印机。这个过程只需要做一次。

    要预先选择打印机：
    • 打开“USB 打印机共享”控制板，点按“网络打印机”标签，然后点按打印机名称旁边的“使用”注记格。
    请为我打开“USB 打印机共享”控制板。

    如果您想使用的打印机不在列表中： 
    • 点按“添加”按钮，打开已共享打印机的网络邻居，选取打印机。

    要在共享 USB 打印机上打印：
    • 使用“选配器”或“桌面打印实用程序”选定打印机，然后像往常一样打印。

疑难解答

    如果您想使用的打印机没有显示在“选配器”或“桌面打印实用程序”中： 
    • 确保已经在您的计算机上安装了打印机的驱动软件。
    • 确保您已使用“USB 打印机共享”控制板来预先选择打印机。

    如果找不到您想预先选择的打印机： 
    • 确保您使用正确的网络邻居。要确认网络邻居，请点按下面的链接。
    设置我的网络邻居。
    • 确保打印机已共享。打开打印机所连接计算机上的“USB 打印机共享”控制板，检查“我的打印机”和“启动/停止”标签。
    • 如果打印机的邻居最近有改变，您的计算机要花费一段时间来检测改变。稍等片刻，然后再试一次。
    • 确保您的计算机使用 TCP/IP 与共享打印机的计算机通讯。您不能通过 AppleTalk 网络共享 USB 打印机。
    • 如果您的计算机与拥有打印机的计算机不在同一个子网，确保在所有中间路由器上启用服务位置协议（Service Location Protocol）多点传送。

    要创建一个“USB 打印机共享”活动性的日志文件： 
    • 打开“USB 打印机共享”控制板，然后点按“选项”按钮，打开“启用记录”。
    记录文件存储在启动磁盘的“系统文件夹”中的“预置”文件夹中。

    要获得 SLP（服务位置协议）怎样注册网络服务（象使用 USB 打印机）的更多信息，请查阅网络服务位置管理器网络管理员指南。要查看这个指南，在 support.info.apple.com/manuals（英文）上搜索“Network Services Location Manager”。
