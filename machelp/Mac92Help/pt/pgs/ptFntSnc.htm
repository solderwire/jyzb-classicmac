匹配字体

    您可以在系统中创建字体描述文件（称为“FontSync 描述文件”），以确保文稿中使用的字体与打印文稿的计算机上的字体匹配。您也可以更改预设字体匹配标准。

创建 FontSync 描述文件

    要创建 FontSync 描述文件：
    • 打开 Apple Extras 文件夹的 Font Extras 文件夹中的 Create FontSync Profile 脚本，然后命名该描述文件。
    请为我打开 Create FontSync Profile 脚本。

    在创建描述文件后，您可以将其拷贝到另一台计算机上，并使用 Match FontSync Profile 脚本将它与该计算机上的字体进行比较。

比较 FontSync 描述文件

    要比较一组字体的 FontSync 描述文件：
    • 请打开 Apple Extras 文件夹的 Font Extras 文件夹中的 Match FontSync Profile 脚本，然后选定要与此计算机上的字体进行比较的 FontSync 描述文件。
    请为我打开 Match FontSync Profile 脚本。
    该匹配脚本将为与描述文件中的字体不匹配的字体创建一个列表。
    提示： 如果您熟悉 AppleScript，可以修改这个脚本来执行其他操作。例如，您可以使该脚本比较两个描述文件。请参阅脚本文件中的注释。
    请为我在“脚本编辑程序”窗口中打开 Match FontSync Profile 脚本。

更改预设的字体匹配标准

    预设的字体匹配标准定义了比较 FontSync 描述文件和计算机上的字体时，应该使用字体的哪些部分。您可以使用 Apple Extras 文件夹的 Font Extras 文件夹中的 FontSync 控制板更改预设的字体匹配标准。要更加方便地使用 FontSync 控制板，请将它移动到硬盘上“系统文件夹”中的“控制板”文件夹。

    注： 对于大多数用户，不需要更改预设的 FontSync 控制板设置。

    要更改预设的字体匹配标准：
    • 请点按创建字体描述文件时想使用的字体标准旁的注记格，然后关闭控制板。
    提示： 要获得更多关于匹配标准的信息，请打开“帮助”菜单并选取“显示旁述”，然后指向您想进一步了解的项目。
    提示： 如果您打算经常使用该控制板，可能要将其放进“系统文件夹”内的“控制板”文件夹中。
