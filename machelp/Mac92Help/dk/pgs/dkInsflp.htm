关于使用磁盘或光盘的问题

读文件的问题

    如果在磁盘上看不到文件：
    您可能正尝试在安装了 Mac OS 8 或更早版本的计算机上使用 Mac OS 扩充格式 (HFS+) 格式化的磁盘。要看到磁盘上的所有文件，必须在计算机上安装 Mac OS 8.1 或更高版本。

    如果看不到 PC SCSI 磁盘：
    使用“文件交换程序”控制板中的“PC Exchange”部分装上磁盘。

    请为我打开“文件交换程序”控制板。

    如果磁盘不可读：
    • 如果磁盘是新的并且没有经过格式化，那么您需要初始化磁盘。
    警告： 初始化磁盘会抹掉磁盘中的全部信息。
    • 磁盘可能已经格式化为在基于 DOS 或 Windows 的计算机上使用。使用“文件交换程序”控制板来打开计算机上的 DOS 或 Windows 文件。
    • 磁盘可能太热或太冷，无法读取。要使磁盘变暖，可以将磁盘在显示器顶部放几分钟。要使磁盘变冷，可以将它放在阴暗、凉爽的地方。
    • 磁盘驱动器可能有问题。按照下列步骤操作：
    1 尝试在磁盘驱动器中使用另一张磁盘。
    2 尝试在另一个磁盘驱动器中使用原始磁盘。
    如果第二张磁盘也不可读，或者原始磁盘在第二个磁盘驱动器上是可读的，那么您的驱动器可能需要修理。

    如果磁盘已损坏：
    应该使用“磁盘修理工具”或“磁盘设置”等程序测试与修理磁盘。

    如果磁盘不能修理：
    您可以初始化磁盘。

    警告： 初始化磁盘会抹掉磁盘中的全部信息。初始化磁盘之前，尽可能地恢复所有信息并将其拷贝至另一个磁盘。

    如果 CD-ROM 光盘图标没有出现在桌面上：
    • 确保光盘标签向上并且光盘放置在托盘的中央。
    • 确保托盘保持关闭。
    • 确保下列功能扩展在“功能扩展”文件夹（系统文件夹内）中：
    • 苹果 CD/DVD 驱动程序
    • 外来文件取用
    • UDF 宗卷取用
    • High Sierra 文件取用
    • ISO 9660 文件取用
    • 音乐光盘取用
    • 苹果照片光盘取用
    使用 Sherlock 程序查看它们是否安装了。
    请为我打开 Sherlock 2。
    如果这些功能扩展在硬盘上，但没有安装到“功能扩展”文件夹，请将它们移到“功能扩展”文件夹，然后重新启动计算机。
    如果找不到它们，您必须重新安装系统软件。
    • 如果您是在购买计算机以后安装的驱动器，确保安装了随驱动器所附的软件。
    • 如果您的计算机有 SCSI 设备，确保没有 SCSI ID 冲突。
    • 检查光盘格式。光盘可能使用了您的计算机不能识别的格式。
    • 重新启动计算机。
    • 从计算机附带的系统软件光盘启动计算机。如果光盘图标没有出现，您的 CD-ROM 驱动器可能有问题。

CD-ROM 驱动器的问题

    如果驱动器中的光盘振动：
    光盘的重量可能有点离心。这不会损坏光盘或计算机。

    如果不能推出光盘：
    确保光盘没有共享。要关闭文件共享，请使用“文件共享”控制板。

    请为我关闭文件共享。
    请为我打开文件共享。

    同时确保光盘上的程序没有打开，文件没有被使用。

    如果驱动器托盘不打开：
    • 选定光盘图标，然后打开“文件”菜单，选取“放回”。或者将图标拖到“废纸篓”。
    注： 如果打开了“简易 Finder”，您需要先将其关闭（在 Finder“编辑”菜单的“预置”窗口中），才能选取“放回”。
    请为我关闭“简易 Finder”。
    • 如果 iTunes 是激活的，请点按应用程序窗口中的“推出”按钮 。

    如果驱动器托盘打不开或者光盘图标没有显示在桌面上：
    • 重新启动计算机。听到启动声音后，立即按下光盘驱动器上的“打开/关闭”按钮。
    • 关闭计算机。在打开的光盘驱动器旁边的小孔中插入拉直的回形针。轻轻地推。

    如果计算机意外推出光盘：
    • 试着重新插入光盘。
    • 光盘可能需要清洗。
    • 光盘或光盘驱动器可能已损坏。按照下列步骤操作：
    1 在驱动器中尝试使用另一张光盘。
    2 尝试在另一个驱动器中使用原始光盘。
    如果第二张光盘也不可读，或者原始光盘在第二个光盘驱动器上是可读的，那么您的驱动器可能需要修理。
