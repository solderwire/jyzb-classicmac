抹掉磁盘内容

    抹掉、重新格式化或初始化磁盘将永久删除磁盘上的内容。抹掉磁盘内容之前，为您不想删除的所有文件作一个备份。

    要抹掉磁盘或可擦除光盘的内容，请使用“其它”菜单中的“抹掉磁盘内容”命令。

    请帮我抹掉磁盘内容。

    注： 在您抹掉磁盘内容时，“格式”弹出式菜单会提供包括“Mac OS 标准格式”（HFS）和 “Mac OS 扩充格式”（HFS+）的选项。“Mac OS 标准格式”与所有版本的 Mac OS 兼容。但是“Mac OS 扩充格式”只与 Mac OS 8.1 或更高版本兼容。如果用“Mac OS 扩充格式”对硬盘格式化，使用 Mac OS 8 或更早版本的计算机不能使用该磁盘。

    要获取可用格式的信息，请从“帮助”菜单中选取“显示旁述”。然后指向您想了解更多的选项。

    如果不能抹掉硬盘内容：
    • 确定这个磁盘不是当前启动磁盘。
    • 确定磁盘没有在网络上共享。要关闭文件共享，请使用“文件共享”控制板。
