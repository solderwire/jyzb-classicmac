使用 CD 与 DVD 光盘

使用 CD-ROM 和 DVD-ROM 光盘

    您可以象使用其它文件一样使用 CD-ROM 光盘上的应用程序和文件。

    您不能写入（存储）任何东西到 CD-ROM 或 DVD-ROM 光盘中。如果想更改光盘上的文件，必须先将他们拷贝到其它类型的磁盘上。

    在 DVD-ROM 驱动器上可以使用 DVD-ROM 和 CD-ROM 光盘。在 CD-ROM 驱动器上不能使用 DVD-ROM 光盘。

使用 DVD-RAM 光盘

    您可以象使用硬盘、软盘或其它活动存储介质一样，使用 DVD-RAM 光盘储存信息。

    您可以使用通用磁盘格式 (UDF)、Mac OS 标准或 Mac OS 扩充格式来格式化 DVD-RAM 光盘。UDF 格式允许您与支持 UDF 1.5 的计算机平台共享信息。

 使用可刻录光盘

    如果您的 Mac 有内建的苹果光盘刻录机，就可以制作自己的光盘。

    CD-R 刻录完成后，文件被“烧制”到光盘上，不能再修改或删除。

    CD-RW（读/写）光盘可以擦除并再次使用。许多旧式计算机不能读取这种类型的光盘，大多数音乐光盘播放程序也与这种类型的光盘不兼容。CD-RW 光盘也会磨损。想要长期储存和稳定性，请使用 CD-R 光盘。
