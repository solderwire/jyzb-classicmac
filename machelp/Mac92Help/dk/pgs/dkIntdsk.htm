初始化磁盘

    使用新磁盘之前，您需要将其初始化（或格式化）。计算机内部的硬盘已在出厂时进行了初始化，因此只在硬盘损坏并且无法使用“磁盘修理工具”（或其它磁盘修理程序）进行修理的情况下，才需要进行初始化。

    警告： 初始化磁盘会抹掉磁盘中的全部信息。初始化磁盘之前，尽可能地恢复所有信息并将其拷贝至另一个磁盘。

初始化 Zip 或 Jaz 磁盘

    要初始化未格式化的 Zip 或 Jaz 磁盘，请使用 Iomega“Tools”程序 （位于硬盘的“Apple Extras”文件夹中）并根据您使用的驱动器类型（例如 USB 或 SCSI）选取适当的实用程序。

    请为我打开“Tools”。

    如果不能打开 Iomega“Tools”程序：
    确保 Zip 或 Jaz 磁盘驱动器已经连接到您的计算机上。如果磁盘驱动器没有连接到计算机上，“Tools”程序可能显示一个错误信息，声明您安装了错误版本的驱动程序。

初始化 DVD-RAM 光盘

    要初始化 DVD-RAM 光盘，按照下列步骤操作：
    1 将磁盘插入 DVD-RAM 驱动器中。
    2 从“其它”菜单中选取“抹掉磁盘内容”。
    3 从“格式”弹出式菜单中选定一种磁盘格式。

初始化硬盘、盒式磁盘或光盘

    要初始化硬盘、盒式磁盘或光盘（不是音乐光盘），请打开“磁盘设置”（位于硬盘的“实用程序”文件夹中）。

    请为我打开“磁盘设置”。

    注： 在您初始化磁盘时，“格式”弹出式菜单会提供包括“Mac OS 标准格式”（HFS）和 “Mac OS 扩充格式”（HFS+）的选项。“Mac OS 标准格式”与所有 Mac OS 版本兼容；“Mac OS 扩充”格式只与使用 Mac OS 8.1 或以后版本的计算机兼容。如果用“Mac OS 扩充格式”格式化硬盘，使用较早版本的 Mac OS 的计算机不能使用该磁盘。

    如果硬盘不是苹果硬盘：
    请使用磁盘制造商提供的（而非“磁盘设置”）实用程序软件。

    如果您要初始化启动磁盘：
    请从系统软件光盘上启动计算机。打开“磁盘设置”，选择启动磁盘，再点按“初始化”按钮。

    如果初始化失败：
    请打开“功能”菜单，选择“初始化选项”，选择“低级格式化”和“将所有数据清零”选项，然后再次尝试初始化磁盘。

    如果初始化再次失败：
    请测试硬盘是否出现物理损坏。

初始化可擦除光盘

    当您插入空白的 CD-RW 或 CD-R 光盘时，会显示一条信息提示您初始化光盘。

    如果 CD-RW 光盘已经包含有文件，您可以将它初始化以再次使用。这会擦除光盘上的所有文件。查看光盘的说明书以了解其使用期限。光盘擦写多次后性能会变得不可靠。

    CD-R 光盘不可以擦写。文件写入到 CD-R 后，不能再被更改或删除。

    要初始化 CD-RW 光盘，请按照下列步骤操作：
    1 将光盘插入光盘驱动器中。
    2 从“其它”菜单中选取“抹掉磁盘内容”。
    3 从“格式”弹出式菜单中选定一种磁盘格式。

    参阅制作光盘可以获得更多信息。
