控制敘述

    通常 AppleScript 會一個接著一個執行敘述。控制敘述能夠更改執行的順序。您可以用控制敘述來略過或是重複執行某些敘述。

    因為控制敘述能夠包含其他控制敘述，所以您可以根據各種不同的狀況來開發繁複的工序指令。

    下列是關於控制敘述的簡單說明，您可以將其用於您的工序指令裡。

Tell 敘述

    Tell 敘述可以指定指令傳送的預設物件。

    tell front window of application "Finder" close

    如果您並未指定預設的物件，AppleScript 會將指令傳給現用的應用程式。

    配合其他的控制敘述，您可以在 Tell 敘述內使用 Tell 敘述。

If 敘述

    底下範例的 If 敘述將會根據情況決定該執行哪一個敘述。在工序指令指令執行時，一個 If 敘述裡至少包含一個非是即偽的表式。

Try 敘述

    指令可能產生錯誤訊息。Try 敘述能夠讓您在錯誤發生時進行控制。Try 敘述會包含錯誤發生時所要執行的敘述。底下是其基本形式：

    try
    [ 敘述 ]
    on error
    [ 敘述 ]
    end try

    如果 Try 之後的敘述執行時發生錯誤，便會接著執行 On Error 子句之後的敘述。

Repeat 敘述

    Repeat 敘述可讓您重複執行敘述。可以利用變數來控制欲重複執行的次數。

    Repeat 敘述有數種形式：
    • Repeat（未確定次數）：敘述重複的次數不確定。
    repeat
    [ 敘述 ]
    end repeat
    在 If 敘述內使用 Exit 敘述，以便停止敘述的重複執行。
    • Repeat 次數：指定重複敘述的次數。
    repeat 整數 [ times ]
    [ 敘述 ]
    end repeat
    • Repeat While：當某個條件成立時重複敘述。
    repeat while表式為真
    [ 敘述 ]
    end repeat
    • Repeat Until：重複敘述直到某條件成立為止。
    repeat until表式為真
    [ 敘述 ]
    end repeat
    • Repeat With 變數：在某數值區間內重複敘述的次數。
    repeat with 迴圈變數 from 起始數值¬
    to 停止數值 [by 間隔數值]
    [ 敘述 ]
    end repeat
    起始數值 和停止數值 決定了變數的區間。間隔數值 決定每次改變的幅度。預設值為 1。
    • Repeat With 列表變數：重複列表所定義的區間次數。
    repeat with 迴圈變數 in 列表
    [ 敘述 ]
    end repeat
    • Exit：停止重複敘述。
