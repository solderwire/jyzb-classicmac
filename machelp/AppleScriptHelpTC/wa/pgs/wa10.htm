敘述

    每個工序指令就是一連串的敘述。每個敘述會包含指令與指令所可能用到的額外資訊。

    敘述可以很簡單，就像這個單一列的指令：

    tell application "Finder" to activate

    敘述通常很複雜，包含多個敘述。底下的範例便包括了多個敘述。

    顯示混合式敘述。

    當您執行工序指令時，每個敘述都依序被執行。然而有些敘述能夠控制執行敘述的順序，如下個範例所示。

    顯示控制型敘述。

    這個範例包含兩個控制型敘述：
    • Tell 敘述控制接收其他敘述的物件。在這個範例當中，Finder 即是該物件。
    • If 敘述控制著第二個 Set 指令是否該執行。只有“item_comments”變數為空值的時候才會執行。
