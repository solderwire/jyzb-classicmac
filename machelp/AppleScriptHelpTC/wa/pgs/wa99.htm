其他功能

    AppleScript 包含了幾個您應該知道的其他功能：
    • 連續字元（¬）：可將一長列打散成數列。
    若要輸入連續字元，請按 Option-Return。
    every file in control panels folder¬
    whose file type is "APPL"
    AppleScript 會把它們當成單一列來處理。
    • 註解：AppleScript 將會忽略的文字。
    註解有兩種形式：
    • （* 屬於區間註解，必須
    放在其他敘述之間。它
    在這個工序指令中可以持續多列。*)
    • quit -- 這是位於敘述末端的註解
    • 識別子：事件名稱，如變數或是屬性標籤或處理元。
    識別子必須以字元起始。
    • 區分大小寫：使用 AppleScript 不必考慮大小寫。
    • 縮寫：AppleScript 能讓您縮寫某些術語。譬如說，您可以在 Tell 敘述內使用“app”而非“application”：
    tell app "Finder" print the front window
    • 選擇性的文字：AppleScript 語言允許您以自然的方式撰寫敘述。有些術語可有可無，譬如前個範例中的“the”。您可以利用它們來幫助您更瞭解工序指令，但對於 AppleScript 來說，它們不是必需的。
