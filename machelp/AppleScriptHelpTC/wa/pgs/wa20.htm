指令

    指令是要讓程式做事的字或片語。底下的範例包含四個指令：

    顯示使用數種指令的工序指令。

    這個範例所包含的指令有：
    • Activate 會將 Finder 變為現用程式。
    • Set 會將最上層的視窗名稱放進容器，在這個工序指令內被命名為“windowName”。這個容器稱為“變數”，因為它可以存放任何名稱。變數是工序指令使用資訊很有用的方式。
    • Close 告訴 Finder 將視窗關閉。
    • Display Dialog 會顯示指定訊息的對話框。

    AppleScript 定義了五個指令：Count、Copy、Get、Set 與 Run。其他程式與工序指令附加程式（位於硬碟內“系統檔案夾”的“工序指令附加程式”檔案夾內）則定義了其他指令。
