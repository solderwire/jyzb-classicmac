檔案夾動作工序指令範例

    底下是由 Finder 所定義的檔案夾動作指令範例。

打開檔案夾

    這個範例包含 Opening Folder 指令處理元。它會取得檔案夾名稱，並會在您將項目放入檔案夾時，顯示對話框告知。

    顯示 Opening Folder 指令。

關閉檔案夾

    這個範例包含 Closing Folder Window For 指令處理元。它會取得檔案夾與磁碟的名稱，然後透過文字轉換語音程式告訴您檔案夾已關閉。

    顯示檔案夾關閉的工序指令。

搬移檔案夾

    這個範例包含 Moving Folder Window For 指令處理元。它會取得原始的與新的視窗位置，並將資訊顯示在對話框內。

    顯示檔案夾被搬移的工序指令。

    每個 Copy 指令都將視窗的位置以四個數字當作一個變數。前兩個數字代表視窗的左上角；後兩個數字代表視窗的右下角。這些數字以像素為單位並以螢幕的左上角為原點。

為檔案夾新增項目

    這個範例包含 Adding Folder Items To 指令處理元。它會取得新增項目的數量與檔案夾名稱，然後透過 Say 指令告訴您相關的資訊。

    顯示為檔案夾新增項目的工序指令。

刪除檔案夾的項目

    這個範例包含 Removing Folder Items From 指令處理元。它會取得檔案夾名稱與項目被刪除的數量，然後透過 Say 指令告訴您相關的資訊。

    顯示刪除檔案夾項目的工序指令。

    這個範例首先將 AppleScript 的文字間隔字符更改為冒
    號（:）。“文字間隔字符”是 AppleScript 用來當作文字區隔的辨識字元。因為“this_folder”參數的檔案路徑名稱是以冒號作區隔，工序指令便可以輕易地辨識出最後一個項目便是檔案夾名稱。

    這個範例也使用到 Try 指令，可讓您處理錯誤情況的發生。因為這個工序指令的 Try 指令之 On Error 部份並未包含任何敘述，所以它將會忽略錯誤訊息。
