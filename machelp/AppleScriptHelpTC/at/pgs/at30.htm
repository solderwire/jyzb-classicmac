總結檔案內容

    假如您手上有數量龐大的文字檔案需要管理，您可能需要每個檔案的內容總結。置於“工序指令附加程式”檔案夾內“標準附加程式”指令詞匯內有可資利用的 Summarize 術語。

    請替我打開“標準附加程式”指令詞匯。

    若要查看 Summarize 指令的語法：
    • 找到 String 指令程式組並按“summarize”一下。

    Summarize 指令需要文字來做總結。您可以提供確切的文字或文字檔的替身，或者是文字檔的檔案說明（包括要打開查看檔案的所有檔案夾名稱）。譬如說這個檔案說明可能像是：

    My Disk:Documents Folder:Text File

    您也許需要這個指令來作檔案的總結，所以工序指令必須能夠讓您自行挑選欲作總結的檔案。底下的範例使用“標準附加程式”指令詞匯內另外兩個指令：
    • 位於 File Read/Write 程式組內的 Read 能夠讀取檔案的內容
    • 位於 User Interaction 程式組內的 Choose File 能夠開啟對話框供您挑選檔案

    顯示總結工序指令。

    這個工序指令可以讓您一次選擇一個檔案，並在“工序指令編寫程式”的“結果”視窗內顯示總結。

    當“工序指令編寫程式”未打開時，您也可以觀看數個檔案的總結，包括那些置於檔案夾內的檔案。底下的範例包含一個 On Open 處理元，您可以將它視為一小段工序指令。若要使用這段工序指令，將它儲存成應用程式，然後在 Finder 中拖動檔案到它上面。這個工序指令可以顯示任何您拖放到它上面的文字檔案的總結。

    顯示可以總結多個檔案的工序指令。
