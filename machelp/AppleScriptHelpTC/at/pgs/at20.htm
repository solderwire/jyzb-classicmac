設定控制面板選項

    在使用某些軟體時，可能得更改顯示器的顏色數目。您可以使用“顯示器”控制面板或“控制條板”上的“顏色數目”模組來更改設定。您也可以透過 AppleScript 來做。

    “系統檔案夾”內的“工序指令附加程式”檔案夾內附“顯示器顏色數目”工序指令附加程式，已事先定義好可用來更改顯示器顏色數目設定的術語。

    若想知道可供使用的術語，請打開“顯示器顏色數目”工序指令附加程式的指令詞匯。

    打開顯示器顏色數目指令詞匯。

    您可以在 System Object Suite 中看到已定義的術語：
    • monitor depth：現用顯示器的顏色數目設定
    • max monitor depth: 最大數量的顯示器顏色數目設定
    • min monitor depth: 最小數量的顯示器顏色數目設定
    • set monitor depth: 設定顯示器顏色數目

    您可以利用這些術語編寫一個可以將顯示器顏色數目設定成您顯示器所能允許的最大顏色數目的簡單工序指令：

    set monitorMaxDepth to max monitor depth
    set monitor depth monitorMaxDepth
