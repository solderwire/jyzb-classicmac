使用收錄來啟動工序指令

    收錄您執行工作時的步驟便可以輕易啟動工序指令。請嘗試製作下列的範例工序指令：
    1 按“工序指令編寫程式”中的“收錄”按鈕。
    2 按一下以啟用 Finder。
    3 打開“編輯”清單，選取“預置…”。
    4 選擇底格空間選項，然後關閉“預置”。
    5 按“停止”按鈕一下。

    當您完成後，便可以在視窗內看見如下的工序指令：

    顯示已收錄完畢的工序指令範例。

    這個工序指令已經可以使用。您如果將它儲存在桌面上，便可以隨時輕易地將“底格空間”預置更改成“疏”的排列方式。

    當工序指令收錄完畢後，您有可能要再作編輯。儘管這個工序指令範例已經可以使用，它的功能卻遠超過於此。所以您可以利用註解符號將用不到的部份“註解”出來：

    顯示編輯範例。

    這個工序指令仍在作用中，因為 AppleScript 並未啟用 Finder，或是打開“預置”對話框以便設定底格空間。

    當然，您也許會想要更改已收錄完成的工序指令以便執行不同的工作。比方說，若想將範例工序指令更改成設定“擠”的“底格空間”的話，可以更改成底下 Set 敘述：

    set uses wide grid of Finder preferences to false
