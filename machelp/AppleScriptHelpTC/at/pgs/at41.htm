參考 Finder 的項目

    若要控制 Finder 的項目（譬如檔案或檔案夾），必須在工序指令內指定項目。參考項目的方法有很多種。您也可以使用 AppleScript 術語指定檔案群組。

參考的方式

    在工序指令內有三個方式可以參考檔案或檔案夾。每種參考方式都會指出項目名稱、項目種類與儲存在電腦內的位置。

    Finder 參考：
    Finder 參考會以檔案或檔案夾的名稱為啟始，然後接著該項目位置的“檔案夾串列”，最後則是磁碟。範例如下：

    file "Art.gif" of folder "Images" of disk "Macintosh HD"

    Finder 參考包括底下特別的檔案夾與磁碟名稱：
    • 系統檔案夾
    • 字體檔案夾
    • 桌面
    • 延伸功能檔案夾
    • 預置檔案夾
    • 蘋果清單項目檔案夾
    • 啟動磁碟

    使用快速鍵時不需要加上引號。底下有兩個範例：

    file "Art.gif" of folder "Images" of startup disk

    file "My Prefs" of preferences folder

    路徑參考：
    路徑參考會以磁碟的名稱為啟始，然後接著所給定的檔案夾名稱，最後則是檔案或檔案夾名稱。每個項目以冒號區分開來。檔案夾的路徑參考則以冒號做結尾。例如：

    file "Macintosh HD:Images:Art.gif"

    folder "Macintosh HD:Images:"

    替身參考：
    替身參考與路徑參考的格式相同，只不過是以“alias”而非“file”或“folder”為啟始：

    alias "Macintosh HD:Images:Art.gif"

    alias "Macintosh HD:Images:"

    替身參考在工序指令內的作用就像 Finder 上的替身一樣。Finder 與路徑參考屬於絕對路徑，所以一旦搬移被參考的項目，工序指令將無法正常運作。如果您是使用替身參考，因為替身能夠包含檔案的位置，所以不管檔案是否被搬移，工序指令都還是可以運作。

    如果您想參考遠端磁碟的項目：
    AppleScript 會在編譯工序指令時，試著打開置於遠端磁碟上的替身參考。遠端磁碟必須是可以取用並且有適當的取用磁碟的使用者名稱與密碼。

貼入參考

    除了直接輸入參考外，您也可以使用拷貝與貼的方式。若要將檔案或檔案夾的 Finder 參考貼到工序指令內：
    1 選擇 Finder 上的檔案或檔案夾，然後打開“編輯”清單並選擇“拷貝”。
    2 回到“工序指令編寫程式”，在工序指令內放置適當的插入點。
    3 打開“編輯”清單並選擇“貼入參考”。

辨識項目

    AppleScript 包含可讓您以不同的方式來指定項目群組的術語。

    Every：
    Every 可讓您參考某個位置內的所有檔案或檔案夾。例如：

    set my_list to every file of folder "Process Files"

    Whose 與 Where Its：
    Whose 與 Where Its 可以參考某些條件下的檔案或檔案夾。例如：

    set my_list to every file of folder "Process Files" ¬whose name contains "project"

    set my_list to every file of folder "Process Files" ¬where its name contains "project"

    這兩個敘述的結果相同。

    這個範例將以它們的標籤屬性來參考檔案夾：

    set my_list to every file of the startup disk whose¬
    label index is not 0

    And 與 Or：
    And 與 Or 可以參考多種條件下的項目。例如：

    set my_list to every file of folder "Process Files" ¬whose name contains "project" and label index is ¬
    not 0

    Entire Contents：
    在 Containers and Folders 程式組內，Finder 定義了包含 Entire Contents 屬性的 Container 物件。因為檔案夾與磁碟都是容器，所以可以使用這個屬性來參考檔案夾內的所有項目。例如：

    set my_list to every file of the entire contents of¬ folder "Process Files" of startup disk whose name ¬
    label index is 0
