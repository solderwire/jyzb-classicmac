檔案夾動作工序指令如何工作

    “標準附加程式”指令詞匯的 Folder Actions 程式組內包含五個指令：

    請替我打開“標準附加程式”指令詞匯。

    每個指令都會傳送事件給 Mac OS 的“檔案夾動作”延伸功能。如果檔案夾所附帶的工序指令包含一個或多個這些指令的處理元，每當有相關連的事件發生時便會開始執行工序指令。

    譬如說，假如您打算在打開檔案夾時執行檔案夾動作工序指令，您可以在工序指令內加入 Opening Folder 指令處理元：

    on opening folder my_folder
    -- statements to execute here
    end opening folder

    每個工序指令可以包含一個或多個這五個指令的處理元，不過每個指令僅能有一個處理元。譬如說，工序指令內可以包含每當檔案夾打開與每當檔案夾關閉的處理元。

    若要將檔案夾動作工序指令附加至檔案夾中：
    1 按住 Control 鍵不放並選擇檔案夾。
    2 從出現的清單中選擇“附上檔案夾動作…”。
    3 在顯示的對話框內選擇工序指令。

    因為檔案夾動作工序指令可以儲存在任何地方，所以您可以將它們保留在“檔案夾動作工序指令”檔案夾內。這個檔案夾包含在 Mac OS 內，就放在“系統檔案夾”的“工序指令”檔案夾中。
