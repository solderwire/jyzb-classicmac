儲存工序指令

    若要儲存工序指令：
    1 打開“檔案”清單，然後選擇“儲存”或“更名儲
    存…”。
    2 使用“格式”清單，選擇工序指令的儲存方式：
    • 已編譯的工序指令程式，只能使用“工序指令編寫程式”來執行
    • 標準 applet
    • Mac OS X 應用程式
    • 文字，可以用其他應用程式來打開
    3 如果您打算將工序指令存成應用程式，請選擇供應用程式使用的選項。
    4 為工序指令命名、選擇欲儲存的位置，然後按“儲存”一下。

    您也可以將工序指令儲存為只供執行。如果工序指令已儲存為只供執行，其他使用者便無法用“工序指令編寫程式”將它打開。

    若要將工序指令儲存為只供執行：
    • 打開“檔案”清單，然後選擇“儲存為只供執
    行…”。

將工序指令儲存為應用程式

    當您將工序指令儲存為應用程式，有時也稱做“applet”，還有幾個儲存方式的選項可供您選擇。

    選擇應用程式的種類：
    您可以將工序指令儲存為標準應用程式，或是 Mac OS X 應用程式：
    • 儲存為標準應用程式，您的工序指令將可在支援 AppleScript 的任何系統上執行。
    • 儲存為 Mac OS X 應用程式，您的工序指令只能在安裝了 CarbonLib（包括 Mac OS X）的電腦上執行。

    儲存為 droplet 應用程式：
    若要執行已儲存為應用程式的工序指令，直接按它兩下即可。如果該工序指令內含 On Open 處理元，您便可以利用將檔案或檔案夾拖放至該應用程式的方式來執行它。這樣的應用程式有時稱為“droplet”。

    選擇選項：
    當您將工序指令儲存為應用程式時，您可以選擇使用這些選項的其中一種：
    • 保持打開
    這個選項可以讓工序指令於執行後保持打開的狀態。若想從其他工序指令發送指令至該工序指令，請選用此項。
    • 永不顯示啟用對話框
    這個選項將不會顯示啟用對話框（顯示工序指令的“描述”欄位內容）。
