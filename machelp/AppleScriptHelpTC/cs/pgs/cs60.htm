撰寫和編輯工序指令

    不管您是從現存的工序指令、收錄動作，或是撰寫自己的工序指令，您都還是需要透過“工序指令編寫程式”做進一步的編輯。在“工序指令編寫程式”內撰寫和編輯工序指令的方式與其他的文字編輯程式沒什麼兩樣。

打散成數列

    您可以使用連續字元（¬）將較長的工序指令列打散成數列。當執行該工序指令時，AppleScript 會把它們當作單一列看待。範例如下：

    make new folder at desktop with properties ¬
    {name:"My Folder", label index:0, selected:true, ¬
    comment:"A folder for my new files!"}

    若要將連續字元插入列中：
    • 請按 Option 與 Return 鍵。

縮排列

    當您撰寫工序指令時，“工序指令編寫程式”會縮排混合的敘述，以便顯示其架構。

    若要縮排混合敘述：
    • 請在每一列尾按 Return 鍵。

格式化工序指令

    當您檢查工序指令的語法時，“工序指令編寫程式”會將工序指令格式化成不同的術語種類。您可以更改工序指令格式化的方式。

    若要更改工序指令格式：
    1 打開“編輯”清單，然後選擇“AppleScript 格式設定…”。
    2 按一下列表內的項目，然後從“字體”和“樣式”清單選擇格式。

    您所作的改變會影響所有打開的工序指令。您無法只做部份的工序指令格式。不過，卻可以指定“描述”欄位內的文字格式。
