使用指令詞匯

    若想知道可以用來控制程式的指令與物件，可以查閱程式的指令詞匯。

    若要打開指令詞匯：
    1 在“工序指令編寫程式”裡打開“檔案”清單，然後選擇“打開指令詞匯…”。
    2 在對話框中找到該程式並打開它。
    3 若要查閱關於術語或程式組的詳細資訊，請按一下將其選取。

    請替我在“工序指令編寫程式”裡打開指令詞匯。

    如果您無法打開程式的指令詞匯：
    該程式無法編寫並且沒有指令詞匯。

讀取指令詞匯

    在指令詞匯裡由相關術語所組成的群組稱之為“程式組”。指令以純文字表示。物件則以斜體字表示。

    若要查看術語或術語程式組的定義：
    • 按一下將它選取。
    定義會出現在視窗的右側。

    定義包含術語（加底線的文字）的簡要說明及其語法。語法是您在工序指令裡編寫術語時應該使用的方式。語法可分成幾個部份。每個部份的說明都會以註解的方式顯示在語法後面。

    指令定義：
    以上面的範例來說，Quit 指令有個簡單的語法。大部份指令的語法裡通常會包含使用該指令時所需的其他資訊。譬如說，Count 指令語法如下：

    count reference -- the object whose elements are to be counted
    each type class -- the class of the elements to be counted
    Result: integer -- the number of elements

    粗體字是在敘述裡使用的術語，必須與指令詞匯裡所使用的完全相同。純文字部份則是您提供的資訊。譬如說，“reference”表示您要計數的物件的名稱。破折號右側的文字屬於說明的部份。

    中括號內的術語可填可不填：

    computer type class -- the attribute to test
    [has integer] -- test specific bits of response
    [Result: integer] -- the result of the query

    物件裡可能包含元件列表，也就是物件內另有物件。元件名稱會以粗體字顯示，之後的純文字則說明在工序指令裡的使用方式。

    物件有其屬性或特性，您可以在工序指令裡使用。屬性的語法會以粗體字表示。屬性之後的純文字則說明該屬性值的種類。有可能是字串、數字、布林值（真或偽），或是參考值。某些情況下，屬性可能會有稱為“常數”的固定值或是常數列表。

    有些屬性的語法包含“[r/o]”。表示該屬性是“唯讀”性質。在工序指令內您可以讀取唯讀屬性，卻無法做設定。
