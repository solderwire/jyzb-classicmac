工序指令的新功能

    此版本的 AppleScript 包含數個新功能，在此加以簡略說明。

    若想知道更多關於這些新功能的資訊，請參觀 AppleScript 網站：www.apple.com/applescript

系統需求

    此版本的 AppleScript（1.5.5 版）需在 Mac OS 9.1或以上版本的系統上執行。其無法在之前版本的 Mac OS 上執行。

Mac OS X 支援

    此版本的 AppleScript 含有下述幾項針對 Mac OS X 的新支援：
    • 除了 Mac OS 9 上任一可工序化的應用程式之外，您還可以在 Tell 區塊中指定在 Mac OS X 上可工序化的應用程式（可和 Mac OS 9 搭配使用）。此外，您也可以指定 Mac OS X Desktop 應用程式。
    如果您將工序指令儲存為一個已編譯的工序指令或標準 applet，您可以透過 Mac OS X Classic 環境來執行這個程式指令，以控制 Mac OS X 上的應用程式或 Desktop 應用程式。
    • “工序指令編寫程式”可顯示在 Mac OS X 上可工序化的應用程式（可和 Mac OS 9 與 Mac OS X Desktop 應用程式搭配使用）所擁有的指令詞彙。
    • “工序指令編寫程式”可將工序指令儲存為 Mac OS X applet。如此您便可以在 Mac OS X 上直接執行工序指令而不必啟動 Classic 環境。
    • 如果 CarbonLib 延伸功能存放在“系統檔案夾”中的“延伸功能”檔案夾裡，您便可以在 Mac OS 8.1 或以上版本的系統上執行 Mac OS X applet。
    • 您可以使用以某個 Mac OS X 應用程式（其內存放有別於“工序指令編寫程式”的工序指令）所製作的工序指令檔案。（此工序指令是存放於檔案中的資料部份，而不是資源部份。）
    如此可讓使用這些工序指令檔案執行某些網路作業更為簡易。

使用 TCP/IP 定址方式控制遠端應用程式

    除了 AppleTalk 名稱可用來辨識電腦之外，工序指令也能夠使用 TCP/IP 位址來控制其他電腦上的應用程式。例如：

    tell application "Finder" of machine "eppc://127.200.126.44"

    您也可以使用電腦的 DNS 名稱：

    tell application "Finder" of machine "eppc://lab.mycompany.com"

    您必須知道電腦的 TCP/IP 位址或 DNS 名稱。您無法透過瀏覽網路的方式找出這些資料。

Using Terms From 指令

    Using Terms From 指令可讓您指定應用程式的專有名詞（不同於 Using Terms From 區塊內 Tell 指令的目標應用程式所使用的專有名稱）。

    此指令可讓您在沒有現用的應用程式可使用時，或無法連接遠端應用程式時，用來開發與編譯工序指令。

    部分範例內容如下：

    using terms from application "Finder"

      tell application remote-Finder
      ...
      end tell

    end using terms from

    AppleScript 會使用 Finder 編譯 Using Terms From區塊內的敘述，而不會更動到 Tell 區塊的目標應用程式。

    您可以在彼此之間加入巢狀式的 Tell 與 Using Terms From 區塊。如果 Using Terms From 區塊在 Tell 區塊內，AppleScript 即會使用 Tell 區塊所指定目標的術語。

改良的 Unicode 文字支援

    AppleScript 可以將由應用程式傳回的 Unicode 文字轉譯成各國的文字，並使用已安裝的任一多語系系統來顯示其結果。

    AppleScript 可以轉譯包含一個以上語系系統的文字。例如，您可以將包含英語、斯拉夫語和日語的文字轉譯成 Unicode 文字，然後再轉譯回各國文字。

    工序指令也可用來比較 Unicode 文字值，以測試其是否相等。

支援大型檔案

    Info For 指令能夠正確地描述檔案、檔案夾以及容量超過 2 GB（十億位元組）之磁碟的大小。

Info For 指令的 Busy Status 屬性

    Info For 指令中的 Busy Status 屬性，當此指令所指定的檔案在使用的狀態時，其會傳回 True 值。

捨去尾數選項

    “標準附加程式”中的 Round 指令有一個“as taught in school”參數，它能讓尾數恰好為 .5 的小數進位至下一個最大值整數。範例如下：

    round 0.5 rounding as taught in school

    如果沒有這個參數，則 Round 指令會將數值進位至最接近的偶數整數。

支援鑰匙圈取用

    如果連接伺服器和卷宗的密碼已經存在於“鑰匙圈取用”時，您便可以使用 Mount Volume 指令來連接 AFP（AppleShare）伺服器卷宗並略去密碼。這樣可以讓您連接網路卷宗而不需將密碼資訊包含在工序指令內。

Folder Action 專有名詞的變更

    Remove Actions From 與 Edit Actions Of 指令（在 Folder Actions 指令詞彙內）的“action number”與“action name”屬性現在已改為“using action number”與“using action name”。

支援 DOS 與 UNIX 文字

    此版本的 AppleScript 能夠識別出 DOS 與 UNIX 系統上文字內容中的分段點。
