关于使用 USB 的问题

    如果 USB 设备无法工作：
    • 如果屏幕上显示信息，指出您没有操作设备所需的必要软件，也许您可以连接 Internet 并自动下载使用设备所需的软件。您也可以与设备制造商联系，以了解怎样获取和安装必要的软件。
    • 如果该设备有根电源线，请确保它是否已插到一个能正常工作的电源插座上并且设备已开启。
    一些 USB 设备（比如打印机），在打开后需预热几分钟。查阅设备附带的手册以获取更多信息。
    • 将设备从 USB 端口拔下并等待几秒钟，然后再将它接回去。
    • 请确保 USB 电缆接头插入正确并且电缆没有毁坏或磨损。
    如果电缆损坏，请更换电缆或者与设备或电缆的制造商联系。

    如果 USB 设备连接到集线器（或者用作集线器的设备）：
    • 如果集线器有根电源线，请确保它已插到一个可以正常工作的电源插座上。
    • 确保集线器的 USB 电缆正确地连接到计算机上。
    • 尝试将设备连接到集线器的其它端口上，或尝试连接其它集线器。
