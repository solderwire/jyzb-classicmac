在哪里可以找到 USB 设备

    目前的 USB 设备包括
    • 打印机和扫描仪
    • 存储设备，例如 Zip、软盘驱动器或 SuperDisk 驱动器
    • 输入设备，例如键盘或鼠标设备
    • 游戏控制器，例如游戏杆、方向盘和游戏手柄
    • 音频设备，例如扬声器和麦克风
    • 通讯设备，例如调制解调器和 ISDN 终端

    重要事项：您不能用 USB 将两台计算机连接到一起。

    查看 USB 产品包装上是否有 Mac OS 标记就可以知道该 USB 产品是否可以在 Macintosh 计算机上工作。您也可以与 USB 设备的制造商联系以查询它是否可以在您的计算机上工作。

    要获得所有 Macintosh 可以使用的 USB 设备的信息，请访问 Macintosh 产品指南网站（网址为 www.macsoftware.apple.com）。
