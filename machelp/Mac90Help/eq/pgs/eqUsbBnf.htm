使用 USB 的主要优点

    下面是 USB 设备的一些突出优点：
    • 您可在计算机开启时插入和拔下设备。
    • 不需要使用终端器或设备 ID 号。
    • 使用 USB 集线器每个 USB 端口最多可以连接 127 个设备。
    某些计算机可能仅支持两个端口加在一起连接最多 127 个 USB 设备。要获得更多信息，请参阅计算机的 USB 规格。
    • 您能够以高速传递信息，计算机上每个 USB 端口最大传输速率为每秒 12 兆位(Mbps)。
    • 设备一旦连接，计算机会自动识别设备。
