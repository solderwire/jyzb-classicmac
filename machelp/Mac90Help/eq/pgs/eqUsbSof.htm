关于 USB 软件

    USB 设备需要称为“驱动程序”的软件才能工作。“驱动程序”是功能扩展文件，安装在硬盘“系统文件夹”的“功能扩展”文件夹中。“驱动程序”可以使计算机在连接 USB 设备时检查该设备，以确保它是否具有足够的电能以及足够的数据接收速度。

    您的计算机已经装有 USB 软件，这样很多第三方 USB 设备一连接到计算机，您就可以使用了。

    如果有以下情况，您可能要安装设备制造商提供的 USB 软件
    • USB 设备不工作
    • 您更想使用设备制造商提供的 USB 软件
    • 设备制造商推荐使用其软件（例如，如果他们更新了 USB 设备的驱动程序）

    要安装驱动程序软件，请按照制造商提供的说明操作。
