验证数字签名

    您可以验证从 Internet 下载的 Mac OS 文件或程序是否来自授权发件者，并可以使用“苹果验证程序”（在“应用程序”文件夹内的“安全”文件夹中）检查其数字签名来验证其未被修改过。

    要验证文件或程序的数字签名：
    • 将项目拖移到“苹果验证程序”图象上。
    请为我显示“苹果验证程序”图象。
    • 如果文件的签名被钥匙串中的鉴定验证过，您可以点按“验证结果”窗口中的按钮来选取打开文件或显示鉴定。
    • 如果签名未被验证，您可以选择信任该签名所说的来自的个人（或组织），该鉴定将被添加到钥匙串。如果您选择不信任，该文件将保留在硬盘上。
    重要事项：在信任鉴定前，您个人应该直接联系该公司以验证其数字“指纹”（显示在鉴定窗口中的该鉴定的唯一 ID 数字）。如果您未验证鉴定的 ID 就选择信任它，可能降低计算机上文件的安全性。

    提示：要获得更多关于“验证结果”或“信任”窗口的信息，请打开“帮助”菜单并选取“显示旁述”，然后指向您想了解的项目。
