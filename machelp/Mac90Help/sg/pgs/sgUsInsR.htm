安装和移走语言

    您可以使用系统软件光盘中的“Mac OS 安装程序”安装和移走不同语言的软件。要添加和移走语言，请执行自定安装，然后选择 Language Kits 选项。

    注意：使用“Mac OS 安装程序”移走语言并不会移走由两种或更多语言使用的文件，也不会移走不是使用安装程序安装的字体。

    如果您要安装阿拉伯和波斯语言软件：
    您不能同时安装用于阿拉伯和波斯语言的软件。要使用波斯语，您必须先移去阿拉伯语软件（如果已安装了阿拉伯语）。要安装波斯语言软件，请参阅系统软件光盘“光盘附加程序”文件夹中 Language Kits CD Extras 文件夹内的 Language Kits CD Extras ReadMe。

    如果在使用与 WorldScript 不兼容的应用程序时遇到问题：
    在重新启动计算机时按住 Option 键和空格键，以暂时停用次语系。

    如果文件或文件夹名显示不正确：
    打开“外观”控制板，点按“字体”标签，选择一种适合您使用语言的字体。例如，如果文件或文件夹名是日语，选择 Osaka 字体。

    如果应用程序菜单和对话框菜单显示不正确：
    将该应用程序拖到 Language Register 程序（位于“实用程序”文件夹），指出使用哪一种语言显示其菜单和对话框。要再次更改这个应用程序的语言，请打开 Language Register，然后选择一个新的语言。您也可以为不同的应用程序和语言建立不同的 Language Register。
