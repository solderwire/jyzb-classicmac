显示不同语言的网页

    您可以使用计算机上或系统软件光盘上的字体，显示不同语言的网页。

中欧语言

    要以中欧语言显示网页，请使用 Courier CE 和 Times CE 字体。这些字体位于系统软件光盘上，要使用它们，需要先进行安装。

    要安装 Courier CE 和 Times CE：
    1 打开 Language Kits CD Extras 文件夹，该文件夹位于系统软件光盘的“光盘附加程序”文件夹中。
    2 连按“Central European.img”图象。
    3 将 Courier CE 和 Times CE 字体拖移到“系统文件夹”的“字体”文件夹中。

古斯拉夫文和中文网页

    要显示古斯拉夫文和简体中文的网页：
    • 按照以下步骤设置 Web 浏览器预置：
    • 对于古斯拉夫文，使用 PriamojProp 和 Latinskij。
    • 对于简体中文，使用 Song 和 Beijing。

    如果中文、日文或韩文的网页包含不可读文本：
    尝试重新载入该页面。在 Web 浏览器中第一次打开时，使用这些语言的页面可能包含不可读的文本。
