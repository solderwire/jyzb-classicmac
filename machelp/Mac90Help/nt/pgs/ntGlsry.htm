Internet 和网络词汇

    操作：“个人 Web 共享”的一个特性，允许万维网文件夹启动一个通用网关接口 (CGI) 程序、非同步 CGI 或其它应用程序。

    访问权限：一个 Mac OS 文件共享特性。访问权限是为单个用户、组别和访客指定的，并决定那些用户是否能看见或更改共享磁盘上的文件。权限在共享项目的“简介”窗口中的“共享…”部分进行设置。

    任何用户：在 Mac OS 文件共享中，“主人”的设置允许任何人访问服务器并更改共享项目。如果在项目“简介”窗口的“共享…”部分中选中了“所有用户”，则任何用户都可以更改共享项目的访问权限。

    苹果远程访问(ARA)：苹果远程访问 (ARA) 用户端软件允许您连接到 ARA 多端口服务器、装有 ARA 个人服务器软件的计算机或者与 ARA 兼容的服务器。

    AppleShare：让您连接到网络上其它计算机和使用共享文件夹或磁盘的内容的软件。您也可以使用 AppleShare 来共享您计算机上的文件夹或磁盘。

    AppleTalk：一种网络协议，由一些网络设备例如打印机和服务器使用。

    浏览器：允许访客查看万维网服务器内容的一种应用程序。

    CGI（通用网关接口）程序：服务器在特定时间运行的应用程序。例如，当一个访客连接到服务器，服务器会打开一个 CGI 程序，它显示一个访客填写的表格，然后把数据发送到一个存储区。

    配置：对特定网络组件（如 AppleTalk 或 TCP/IP）的一组设置。配置划分为不同的组，这样您可以切换组来切换属于该组的所有配置。

    域类型：确定网站所含内容的类型或者网站与何种组织有关的网站后缀（例如用于商业方面的“com”, 用于教育部门的“edu”, 用于政府机关的“gov”, 用于非赢利组织的“org”）。

    所有用户：适用于除“主人”和“用户/组别”的所有用户（注册的用户和客人）的 Mac OS 文件共享访问权限类别。“所有用户”的权限在一个项目的“简介”窗口的“共享…”部分进行设置。

    “文件共享”控制板：用于打开或关闭文件共享的控制板。文件共享允许网络上的注册用户访问共享的文件夹和文件。

    客人：没提供名称连接到服务器的任何人。客人权限在“文件共享”控制板中的“用户与组别”标签下设置，可在 Mac OS 文件共享中打开或关闭。

    主页：当访客连接到万维网服务器时显示的第一个 HTML 文稿。

    HTML (超文本标记语言)：一种处理文本文件的方法，可让它们能显示内容并连接到万维网上的其它文件。

    Intranet（企业内部互连网）：基于网络的万维网服务器和网站的集合，通常位于一个公司或其它组织的内部。

    Internet：由连接万维网网站和其它服务（如电子邮件，讨论组和 FTP 服务）的计算机及通讯软硬件组成的大型全球网络。

    Internet 服务供应商 (ISP)：提供直接连接到 Internet 的公司。ISP 通常拥有快速的、专用的 Internet 连接服务和单个用户通过电话线连接的多个调制解调器。在用 ISP 建立帐号时，这个公司会提供给您为 Internet 连接而设置计算机所需的信息和基本软件。

    IP 地址：IP 是 Internet 协议的缩写，一种指定信息如何通过计算机网络进行传送的方法。IP 地址，它标识一台特定的计算机，由 4 到 12 个被英文句号分成四部分的数字组成（例如 130.43.2.2）。

    MIME 类型：与万维网服务器上的文件关联的信息。MIME 类型告诉服务器和任何已连接的用户的万维网浏览器此文件的类型和应该使用哪个应用程序来打开它。（MIME 是 Multipurpose Internet Mail Extension 的首字母缩写。）

    调制解调器：使您的计算机能通过普通电话线与其它计算机进行通讯的设备。调制解调器将数字式信息转换成大多数电话线上使用的模拟信号。

    网络：使用专门的软件和硬件互相连接以共享和传送数据的一组计算机。

    主人：Mac OS 文件共享中控制指定文件夹的人。

    个人 NetFinder：“个人 Web 共享”的一项特性，可将万维网文件夹的内容显示成一个列表。

    端口：表示两个设备通讯用的级别的数字；通讯过程中每个设备的级别必需一样。例如，万维网服务器和浏览器使用 HTTP 通讯，它运行在 80 端口上。在“个人 Web 共享”中, 端口是您的计算机上的万维网文件夹的地址。

    注册用户：在 Mac OS 文件共享中，与该名称有关的，具有访问权限的那个人的名称。

    TCP/IP 网络连接：Transfer Control Protocol/Internet Protocol（传输控制协议／Internet 协议）的缩写，是指定信息如何通过 Internet（通过网络连接）传送的方法。

    URL（统一资源定位器）：网站或其它 Internet 服务的“地址”或位置 。

    用户与组别: 在 Mac OS 文件共享中，“文件共享”控制板中用来指定注册用户、组和客人的一个标签。

    网页：在万维网服务器上的任何 HTML 文件。

    万维网服务器：运行通过 Internet 或 Intranet 共享 HTML 和其它文件的软件的计算机。

    “Web 共享”控制板：您用来设置“个人 Web 共享”的软件。

    网站：收集了许多 HTML 文件和其它内容，访客能通过 URL 访问并用万维网浏览器浏览。
