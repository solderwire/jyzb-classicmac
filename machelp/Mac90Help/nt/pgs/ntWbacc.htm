控制对万维网文件夹的访问

    当您使用“个人 Web 共享”来共享文件或发布网页时，可以控制谁可以访问和怎样使用该文件。

    预设情况下，所有的用户都有“只读”访问权；也就是说，他们能看见文件，但不能对其进行删除、替换或者更改。

为 Web 文件夹中的项目分配访问权限

    要为 Web 文件夹中的项目分配访问权限：
    1 请打开“Web 共享”控制板。
    请为我打开“Web 共享”控制板。
    2 点按标有“使用‘文件共享’来控制用户访问”的按钮。
    3 为整个 Web 文件夹或单个文件和 Web 文件夹中的文件夹设置访问权限。

    重要事项：如果使用文件共享控制对您的 Web 服务器的访问，在 Web 服务器运行时必须打开文件共享。如果关闭文件共享，除了您以外没有其它人可以访问您的文件。（在打开文件共享后访问权限会被恢复。）
