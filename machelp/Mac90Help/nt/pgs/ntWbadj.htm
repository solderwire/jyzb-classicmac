调整“个人 Web 共享”预置

    您可以指定某些“Web 共享”选项，例如“Web 共享”活动是否会被记录，内存分配，“Web 共享”端口编号，MIME 类型，操作和其它预置。

    要调整“个人 Web 共享”设置，请在“Web 共享”控制板中打开“编辑”菜单并选取“预置…”。

    请为我打开“Web 共享”控制板。

    提示：要获得关于各种设置的信息，请打开“帮助”菜单并选取“显示旁述”，然后指向您感兴趣的项目。

    要显示“个人 Web 共享”的完整指南：
    • 请使用万维网浏览器来显示硬盘上“Web 页面”文件夹中的文件“Using Personal WebSharing.html”。
