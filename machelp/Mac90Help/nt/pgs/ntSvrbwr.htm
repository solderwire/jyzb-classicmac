使用“网络浏览器”

    您可以使用“苹果”菜单中的“网络浏览器”连接到 AppleShare、FTP 或 HTTP 服务器。

    请为我打开“网络浏览器”。

    要连接到服务器：
    • 在列出的网络邻域中导航以找到该服务器，然后连按其名称。
    提示：您可以使用向后和向前箭头来反向执行您执行过的步骤，或点按“迅捷”按钮并选取“网络”，以返回出发点（邻域列表的最高一层）。

    要使用 URL 或 IP 地址连接到服务器：
    • 点按“迅捷”按钮并选取“连接到服务器…”，然后键入该服务器的 IP 地址或 URL 并点按“连接”。
      “迅捷”按钮

    要连接到最近访问过的服务器：
    • 点按“最近使用”按钮并从弹出式菜单中选择服务器的名称。
      “最近使用”按钮

    要连接到您喜欢的服务器或区域：
    • 点按“个人收藏”按钮并从弹出式菜单中选取该服务器的名称。
      “个人收藏”按钮

    要将服务器或邻域添加到收藏列表：
    • 选择列表中项目的名称，然后点按“个人收藏”按钮并从弹出式菜单中选择“加入个人收藏”。
    您也可以将项目从列表中拖移到“个人收藏”按钮上。

    提示：要为服务器或邻域建立替身，请将项目名称拖移到桌面上。

    如果您寻找的服务器没有列出来：
    • 检查 AppleTalk 和 TCP/IP 控制板中的配置。
    • 检查 TCP/IP 控制板中指定的搜索域。
    • 检查 Internet 控制板中的“高级”标签的“主机”设置中的指定 LDAP（简单目录访问协议）服务器。
    • 询问您的网络管理员，确定该服务是否已在您计算机可以看到的网络段目录中注册。
    • 参阅“网络服务位置管理程序网络管理员指南”以了解应用程序如何找到网络服务，以及您或您的网络管理员如何使特定的服务显示在服务列表中。您可以在 support.info.apple.com/manuals 网站找到指南。
