使用 SNMP 管理网络

    您可以使用 SNMP Administrator（由系统软件光盘提供）通过简单网络管理协议（SNMP）设置和监视网络上的计算机组。

    请为我打开 SNMP Administrator。

    如果 SNMP Administrator不在您的硬盘上：
    没有安装 SNMP Administrator。要安装 SNMP Administrator，请使用随计算机附带的 Mac OS 光盘或软件安装光盘中的 Mac OS 安装程序。（该程序位于 光盘附加程序: Network Extras: OTExtras 文件夹内。）

要获得更多的信息

    要获得关于 SNMP 网络管理的信息，请查阅 SNMP 管理应用程序的电子手册，SNMP Admin User's Guide。该指南在随计算机提供的 Mac OS 光盘或软件安装盘的“Network Extras”文件夹里。
