发布网页

    您可以使用“个人 Web 共享”，从硬盘上的文件夹在 Internet（或者在公司或学校的内部互联网）上发布网页或共享文件。

    要发布网页：
    1 将 HTML 格式的文件拷贝到硬盘上的“Web 页面”文件夹。
    2 如有必要，请连接到 Internet 上。
    3 使用“Web 共享”控制板指定其中一个文件作为主页。主页是访客访问网站时看到的第一个页面。
    请为我打开“Web 共享”控制板。
    4 使用“Web 共享”控制板启动“Web 共享”。

    重要事项：放进“Web 页面”文件夹的文件必须是 HTML 格式，因为其他用户要将其以网页的形式显示。

    要共享任何类型的文件：
    • 在“Web 共享”控制板中选定“无”作为主页。您的文件将由“个人的 NetFinder”列在其他用户的浏览器窗口中。
    用户可以通过将文件名称从这个列表拖移到用户的桌面或其它文件夹来拷贝文件。

    要在显示 HTML 页的同时也允许其他用户从列表拷贝文件，需要设置“个人的 NetFinder”链接：
    • 在“Web 共享”控制板中指定一个主页，然后将一个链接添加到其中一个格式中含有“PNF:”的 HTML 页面中：
    http://服务器的名称或 IP 地址/PNF:
    http://服务器的名称或 IP 地址/PNF:子文件夹/

    要显示“个人 Web 共享”的完整指南：
    • 请使用万维网浏览器来显示硬盘上“Web 页面”文件夹中的文件“Using Personal WebSharing.html”。
