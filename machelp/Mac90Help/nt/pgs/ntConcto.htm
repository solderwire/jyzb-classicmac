选择网络服务

    某些应用程序允许您打开特定的 URL 或浏览网络，以使用应用程序特定或一般服务（如 AppleShare、FTP 或 HTTP）。

    要浏览服务：
    • 请展开列出的网络邻域直到您看到该服务，点按以选定它，然后点按“选取”。

    要选定您寻找的服务器类型：
    • 请点按“迅捷”按钮并选定服务类型。

    如果您寻找的服务没有列出来：
    • 请检查所选的服务类型（例如 Web 服务器或 FTP 服务器）。
    • 检查在 TCP/IP 控制板中指定的搜索域。
    • 检查 Internet 控制板中“高级”标签的“主机”设置中的指定 LDAP（简单目录访问协议）服务器。
    • 询问您的网络管理员，确定该服务是否已在您计算机可以看到的网络段目录中注册。
    • 参阅“网络服务位置管理程序网络管理员指南”以了解应用程序如何找到网络服务，以及您或您的网络管理员如何使特定的服务显示在服务列表中。您可以在 support.info.apple.com/manuals 网站搜索该标题来查找该指南。

    提示：当您打开文件对话框时，按住 Control 键以查看服务器列表中附加到网络邻域的源目录类型。
