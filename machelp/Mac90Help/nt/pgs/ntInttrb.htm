关于连接 Internet 的问题

关于用调制解调器连接的问题

    如果不能连接到 ISP：
    ISP（或称 Internet 服务提供商）是为您提供连接到 Internet 服务的公司。如果有关于 Internet 帐户的问题，请与 ISP 联系以获得帮助。

    如果使用“设置助理”来登记 Internet 帐户，关于您的 ISP 的信息将存放在硬盘上 Internet 文件夹中的“我的 Internet 设置”文稿中。

    请为我打开“我的 Internet 设置”。

    请确保 Internet 控制板中的信息与“我的 Internet 设置”文稿中的信息相匹配。

    请为我打开 Internet 控制板。

    如果找不到“我的 Internet 设置”：
    该文件可能被移走或更名，或者您可能并未使用“设置助理”配置 Internet 设置。

    如果可以连接 ISP 但是浏览器不显示任何网页：
    请确保您已在 TCP/IP 控制板中选择了正确的 TCP/IP 设置。

    如果您想监视连接尝试：
    使用“苹果”菜单中的“远程访问状态”程序连接。

    如果您不能用 Internet 拨号程序来进行连接：
    Internet 拨号程序可能是使用 Mac OS 的较早版本安装的。Internet 拨号程序不再与 Mac OS 兼容，不应用于建立连接。

    如果找不到 DNS 服务器：
    找不到您输入的地址，或者您的计算机没有连接到 Internet。请确保地址是否正确并且您的连接是否处于现用状态。

    如果计算机不能建立网络插座连接：
    计算机不能连接到 Internet。请确保您的调制解调器设置正确并已经打开，以及您在 TCP/IP 控制板中是否已选择正确的 TCP/IP 配置。

关于通过网络连接的问题

    如果不能完成网络连接（没有网页显示）：
    请确保您已在 TCP/IP 控制板中选择了正确的 TCP/IP 设置。

    如果找不到 DNS 服务器：
    找不到您输入的地址，或者您的计算机没有连接到 Internet。请确保地址是否正确并且您的连接是否处于现用状态。

    如果计算机不能建立网络插座连接：
    计算机不能连接到 Internet。请确保您已在 TCP/IP 控制板中选择了正确的 TCP/IP 设置。
