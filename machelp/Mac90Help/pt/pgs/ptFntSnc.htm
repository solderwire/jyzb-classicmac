匹配字体

    您可以在系统中建立字体描述文件（称为“FontSync 描述文件”），以确保文稿中使用的字体与打印文稿的计算机上的字体匹配。您也可以更改预设的字体匹配标准。

建立 FontSync 描述文件

    要建立 FontSync 描述文件：
    • 打开 Apple Extras 文件夹的 Font Extras 文件夹中的 Create FontSync Profile 命令语言，然后命名该描述文件。
    请为我打开 Create FontSync Profile 命令语言程序。

    在建立描述文件后，您可以将其拷贝到另一台计算机上，使用 Match FontSync Profile 命令语言来将其与该计算机上的字体加以比较。

比较 FontSync 描述文件

    要比较一组字体的 FontSync 描述文件：
    • 请打开 Apple Extras 文件夹的 Font Extras 文件夹中的 Match FontSync Profile 命令语言，然后选定要与此计算机上的字体加以比较的 FontSync 描述文件。
    请为我打开 Match FontSync Profile 命令语言程序。
    该匹配命令语言将建立一个描述文件中与该计算机字体不匹配的字体列表。
    提示：如果您熟悉 AppleScript，可以修改该命令语言以执行其它操作。例如，您可以使该命令语言比较两个描述文件。请参阅命令语言文件中的注释。
    请为我在“命令语言编辑程序”窗口中打开 Match FontSync Profile 命令语言程序。

更改预设的字体匹配标准

    预设的字体匹配标准定义了比较 FontSync 描述文件和计算机上的字体时，应该使用字体的哪个部分。您可以使用 Apple Extras 文件夹的 Font Extras 文件夹中的 FontSync 控制板更改预设的字体匹配标准。

    请为我打开 FontSync 控制板。

    注意：对于大多数用户，不需要更改预设的 FontSync 控制板设置。

    要更改预设的字体匹配标准：
    • 请点按建立字体描述文件时想使用的的字体标准旁的注记格，然后关闭控制板。
    提示：要获得更多关于匹配标准的信息，请打开“帮助”菜单并选取“显示旁述”，然后指向您想进一步了解的项目。
    提示：如果您打算经常使用该控制板，可能要将其放进“系统文件夹”中的“控制板”文件夹里。
