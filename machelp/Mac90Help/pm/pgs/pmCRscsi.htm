解决 SCSI ID 冲突

    如果您的计算机配备有 SCSI（小型计算机系统接口）设备，应确保每个设备具有介于 0 到 7 之间的一个唯一 ID 号，通常内置 SCSI 设备具有以下 ID 号：

      内置设备   ID    
    硬盘 0
    CD-ROM 驱动器 3
    Zip 驱动器 5
    Macintosh 计算机 7

    注意：有些计算机具有两条独立的 SCSI 链，每条链具有各自一组 ID 号。有些计算机配有 ATA 或 IDE 硬盘驱动器，它们都没有 SCSI ID 号。请参阅随计算机附带的说明和技术信息。

    请确保 SCSI 链端接正确。通常每个 SCSI 链的第一个和最后一个设备必须端接。

    请参阅随计算机和 SCSI 设备附带的使用手册以获得设置 SCSI ID 号和端接 SCSI 设备的具体说明。
