关于 Mac OS 9
    这部分提供了关于 Mac OS 9 的最新信息。
    安装以后显示器分辨率发生改变
    在执行 Mac OS 9 独立安装后，显示器的分辨率会与原来的设置不同。您可以使用“显示器”控制板改回原来的显示器分辨率。
    Iomega 驱动程序和虚拟内存
    Mac OS 9 允许您将可装卸式介质（如 Iomega Jaz 驱动器）用于虚拟内存。

    如果要将 Mac OS 9 安装到 Zip 或 Jaz 磁盘，应该将不可装卸的后备宗卷用于虚拟内存。要这样做，请在从“其它”菜单选择“重新启动”时按下 Shift 键（停用“功能扩展”）。然后打开“存储器”控制板并打开虚拟内存。选择一个与虚拟内存兼容的宗卷，作为虚拟内存宗卷。
    磁盘设置和节能器
    如果需要使用“磁盘设置”来执行占用时间超过几分钟的操作，例如测试磁盘或低级格式化磁盘，请务必在“节能器”控制板中将滑杆拖到“无”，以使硬盘不进入睡眠。
    使用 Acrobat PDF 文稿
    一些 Mac OS 9 附带的文稿是使用 Adobe Acrobat 建立的（这些文件的名称以“pdf”结尾）。要浏览或打印这些文稿，必须安装 Adobe Acrobat Reader（位于 Mac OS 9 光盘中“光盘附加程序”文件夹的“Adobe Software”文件夹中）。
    Language Kits
    由自定安装（在 Mac OS 安装器的“自定安装”窗口中选择“Language Kits”）的 WorldScript I 语言（即阿拉伯语、梵文、古吉拉特语、旁遮普语和希伯来语）并非大多数的 Internet 应用程序都完全支持。在网页上以这些语言显示的内容中会看见一些不可读的文本。

有关兼容性的注释

    苹果 CD-ROM 功能扩展和非苹果 CD-ROM 的不兼容性
    如果计算机或 CD-ROM 驱动器不是由苹果公司制造的，或者您有苹果 CD-ROM 驱动器，而使用的是非苹果驱动程序，则 CD-ROM 驱动器可能无法正常工作。从“功能扩展”文件夹中（在“系统文件夹”中）移走名为“苹果 CD / DVD 驱动程序”的文件，并重新安装随 CD-ROM 驱动器附带的软件。请联系 CD-ROM 的制造商，以查看您是否有他们软件的最新版本。
    Apple HD SC Setup 程序
    在安装 Mac OS 9 后，不要使用 Apple HD SC Setup 程序初始化苹果硬盘。应使用 Mac OS 9 光盘上的“磁盘设置”程序来初始化。
    用于安装 QuickTime 的应用程序
    一些应用程序以较早的版本替换了由 Mac OS 9 安装的 QuickTime 版本。如果出现这种情况，需要从 Mac OS 安装器中选择“增加/移去”来重新安装 QuickTime，然后进行自定安装。要获得更多信息，请搜索“增加或移去系统软件”。
    Connectix Virtual PC
    Connectix Virtual PC 2.1.1 和更早的版本与 Mac OS 9 不兼容。如果您在使用 Connectix Virtual PC，请确定所使用的是版本 2.1.2 或最高的版本。您可以在 Connectix 网站下载 2.1.2 更新程序（适用于 Virtual PC 2.0 用户）： http://www.connectix.com/html/updates.html 或拨打 1-800-950-5880 联系 Connectix 销售部门，以获得有关升级版的信息。
    Dr. Solomon's Virex
    如果您在使用 Dr. Solomon's Virex 程序版本 5.9 或更早的版本，请务必关闭“Scan Files When Opened”选项。该选项与 Mac OS 9 不兼容。要获得关于升级到 Virex 5.9.1 的更多信息，请使用以下信息与 Dr. Solomon's 软件公司联系：virex-tech@drsolomon.com 或 http://www.drsolomon.com/
    FM Radio 应用程序和外观声音
    在打开外观声音（在“外观”控制板中）后打开 FM Radio 应用程序可能会停用计算机的声音。要修正这个问题，在打开 FM Radio 应用程序前关闭外观声音，然后在该应用程序打开后再打开外观声音。
    File Synchronization 控制板
    如果控制板窗口中列表的文件夹或宗卷有自定的图象，File Synchronization 控制板可能不能正常工作。要从文件夹或宗卷中移走自定的图象，选择桌面上的图象，打开“文件”菜单，选择“简介”。然后选择窗口左上角的的图象，打开“编辑”菜单选择“剪”。
    苹果远程访问个人服务器
    如果您正在使用“苹果远程访问个人服务器”，执行 Mac OS 9 的独立安装，在完成安装第一次重新启动计算机后，服务器可能不能正确应答进入的呼叫。在使用“苹果远程访问个人服务器”前再次重新启动计算机。
    AppleShare 4.0.2
    您不能拷贝文件到位于 AppleShare 4.0.2 服务器的拖放框中。这个问题不会出现在 AppleShare 更新的版本中。
    Adobe Type Manager 和 Adobe Type Reunion
    Mac OS 9 中的更改产生与 Adobe Type Manager (ATM)、 ATM Deluxe 和 Adobe Type Reunion Deluxe (ATR) 的不兼容性。在安装过程中 Mac OS 9 安装器自动将 ATM 控制板移到“控制板（停用）”文件夹中。Adobe 公司正在准备 ATM 4.5.2、ATM Deluxe 4.5.2、ATR Deluxe 2.5.2 的更新，以解决这些问题。这些更新程序将很快可以从 http://www.adobe.com/support/downloads/main.html 获得。

关于 PowerBook 计算机的注释

    密码保护只能保护一个分区
    使用“密码保护”控制板以要求输入一个密码来启动您的便携式计算机。如果已经格式化内置硬盘以使其具有多个分区，只有启动部分的分区会有密码保护。另外，如果有多个分区包含“系统文件夹”，计算机可能会从没有保护的分区启动。
    PowerBook 3400/G3 Modem
    要使用 PowerBook 3400 或 PowerBook G3 以太网必须安装“PowerBook 3400/G3 Modem”功能扩展。该功能扩展由 Mac OS 9 安装器安装，即使安装有其它调制解调器软件，也不应将其移走。
    控制条和 Zip 磁盘
    在某些型号的 Powerbook 上，Zip 磁盘不会出现在“介质板”控制条模组中，但是会出现在桌面上。

关于打印的注释

    在 LocalTalk 网络和串行打印机之间切换
    如果您使用同一端口交互连接 LocalTalk 网络或串行打印机，在关闭 AppleTalk 以使用打印机时端口可能无法正确重置。如果在关闭 AppleTalk 后打印有问题，请重新启动计算机。
    重新安装打印机驱动程序
    如果执行了 Mac OS 9 的独立安装，而在选配器中没有打印机图象，则可能需要重新安装随打印机附带的软件。

    • 如果您使用的是 StyleWriter、StyleWriter II 或 StyleWriter 1200，请安装随 Mac OS 9 附带的 Color SW 1500 驱动程序（而不要安装原始打印机软件）。

    • 如果您使用的是 Color StyleWriter 2200 或 2400，请安装随 Mac OS 9 附带的 Color SW 2500 驱动程序（而不要安装原始打印机软件）。

    您可能还需要重新安装随打印机软件附带的字体。请按照随打印机软件附带的说明进行。
    Apple LaserWriter 8f 打印机驱动程序
    LaserWriter 8f 打印机驱动程序与 Mac OS 9 不兼容。如果已经安装了该驱动程序，将其移去，使用 LaserWriter 8 驱动程序。
    Color StyleWriter 4000 和 DeskJet 600 打印机
    苹果 Color StyleWriter 4000 系列和 Hewlett-Packard DeskJet 600 系列打印机用在 Mac OS 9 时要求更新的打印机驱动程序。苹果公司和 HP 公司正在进行这方面的工作，以提供修正这个问题的驱动程序。该驱动程序将很快可以在苹果公司和 HP 公司的网站上获得。

关于“多用户”环境的注释

    使用 Epson 打印机
    要在 Epson 打印机上打印，在“受限制”或“面板”环境下的用户应该配置 Epson 打印机的驱动程序，来使用位于用户文件夹中的脱机打印文件夹或用户可以创建文件的其它位置。在 Epson 打印机对话框中，点按“工具”按钮，然后点按“设置”。再改变临时脱机打印文件夹和临时高速拷贝文件夹的位置。
    在“多用户”环境下的打印机访问
    不出现在桌面上的打印机不会显示在“多用户”控制板的可用打印机列表中。要让受限制的用户使用这些打印机，则要允许访问“所有打印机”或允许用户访问“选配器”。
    PowerBook 5300 和语声密码
    如果是 PowerBook 5300 计算机，则不能使用语声验证，必须使用一个键入的密码。
    程序兼容性
    如果您在“受限制”或“面板”用户环境下遇到问题，请在“Mac 帮助”中搜索“多用户”，查找疑难解答和其它关于配置软件的重要信息。

各方面的注释

    iMac Update 1.0 和 Mac OS 9
    iMac Update 1.0 对装有 Mac OS 9 的 iMac 不是必要的。如果有更新就不要再使用了。这样做可能会造成在计算机重新启动时出现“死机”。
    听音乐光盘
    带有 ATA 光盘或 DVD 驱动器的计算机现在使用数字音乐光盘回放，以提高逼真度和与数字音频周边设备的兼容性。在这种计算机上听音乐光盘，不再需要在“声音”控制板或“控制条”中选择光盘作为输入源。要从光盘收录，需要选择内置光盘作为音频源。
    Macintosh Manager
    请参阅技术信息库的章节，“Mac OS X Server: Macintosh Manager and Mac OS 9”获取最新的信息。访问技术信息库的网址是 http://til.info.apple.com/
