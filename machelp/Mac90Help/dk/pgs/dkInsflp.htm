关于使用磁盘或光盘的问题

关于读取文件的问题

    如果您不能看到磁盘上的文件：
    您可能正尝试在使用 Mac OS 8 或以前版本的操作系统的计算机上使用“Mac OS 扩充格式（HFS+）”格式化过的磁盘。要想看到该磁盘上的所有文件，您需要在装有 Mac OS 8.1 或以后版本操作系统的计算机上使用该磁盘。

    如果您不能看到 PC SCSI 磁盘：
    使用“文件交换程序”控制板的“PC Exchange”部分来装上磁盘。

    请为我打开“文件交换程序”控制板。

    如果磁盘不可读：
    • 若是新磁盘且未格式化，则您需要初始化磁盘。
    警告：初始化磁盘会抹掉其中的全部信息。
    • 磁盘可能已在基于 DOS 或 Windows 的计算机上进行了格式化。使用“文件交换程序”控制板来打开计算机上的 DOS 或 Windows 文件。
    • 磁盘可能温度太高或太低，导致无法读取。要对磁盘升温，可以将它在显示器顶部放几分钟。要给磁盘降温，应将其放在阴凉的地方。
    • 您的磁盘驱动器可能出现问题。请执行下列步骤：
    1 尝试在磁盘驱动器中使用另一磁盘。
    2 尝试在另一磁盘驱动器中使用原磁盘。
    如果第二个磁盘也不能读取，或原磁盘在其它磁盘驱动器中工作正常，则说明您的磁盘驱动器可能需要维修。

    如果磁盘损坏：
    您需要使用一些程序（如“磁盘修理工具”或“磁盘设置”）来测试与修理磁盘。

    如果无法修理磁盘：
    您可以初始化磁盘。

    警告：初始化磁盘会抹掉其中的全部信息。初始化磁盘之前，尽可能地恢复所有信息并将其拷贝到另一个磁盘。

    如果桌面上没有显示 CD-ROM 光盘图象：
    • 请确保光盘带标签的一面朝上，光盘完全放入托盘。
    • 请确保托盘完全关闭。
    • 确保下述功能扩展在功能扩展文件夹（在“系统文件夹”内）中：
    • 苹果 CD/DVD 驱动程序
    • 外来文件取用
    • UDF 宗卷取用
    • High Sierra 文件取用
    • ISO 9660 文件取用
    • 音乐光盘取用
    • 苹果照片光盘取用
    使用 Sherlock 程序查看它们是否已安装。
    请为我打开 Sherlock 2。
    如果这些功能扩展在您的硬盘上而没有安装到“功能扩展”文件夹中，请将其移动到该文件夹中，然后重新启动计算机。
    如果您找不到它们，则需要重新安装系统软件。
    • 如果您是在购买计算机后才安装驱动器的，应确保安装随驱动器提供的软件。
    • 若您的计算机上连接有 SCSI 设备，应确保不存在 SCSI ID 冲突。
    • 检查光盘格式。光盘可能使用了计算机无法识别的格式。
    • 重新启动计算机。
    • 请从随计算机提供的系统软件光盘启动计算机。若光盘图象没有出现，则可能是 CD-ROM 驱动器出现了问题。

关于 CD-ROM 驱动器的问题

    若光盘在驱动器中振动：
    光盘的重心可能稍微错位。这不会损坏光盘或计算机。

    若不能推出光盘：
    确保光盘未被共享。要关闭文件共享，请使用“文件共享”控制板。

    请为我关闭文件共享。
    请为我打开文件共享。

    另外请确保光盘上的程序没有打开或文件没有在使用。

    若驱动器托盘打不开：
    • 请选择光盘图象，打开“文件”菜单并选取“放回”，或者将图象拖至“废纸篓”。
    注意：如果打开了“简易 Finder”，您需要先将其关闭（在“编辑”菜单的 Finder 的“预置…”窗口），才可以选取“放回”。
    请为我关闭“简易 Finder”
    • 若“苹果音乐光盘程序”处于现用状态，请打开“文件”菜单，选取“推出光盘”，或按一下“苹果音乐光盘程序”窗口中的推出按钮。

    若驱动器托盘打不开，并且桌面上没有显示光盘图象：
    • 重新启动计算机。听到启动声音时，按驱动器上的“打开/关闭”按钮。
    • 关闭计算机。将拉直的曲别针插入驱动器开口处旁边的小孔中。再轻轻推动。

    若计算机意外推出光盘：
    • 请尝试重新插入该光盘。
    • 光盘可能需要清洁。
    • 光盘或驱动器可能已损坏。请执行下列步骤：
    1 在该驱动器中试用另一个光盘。
    2 在另一个驱动器中试用原光盘。
    若第二个光盘也不能读取，或原光盘在其它驱动器中工作正常，则说明您的驱动器可能需要维修。
