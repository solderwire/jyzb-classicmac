抹掉磁盘内容

    抹掉磁盘，重新格式化或初始化磁盘会永久性地删除磁盘上的所有内容。抹掉磁盘之前，请建立不想抹掉的所有文件的拷贝。

    要抹掉磁盘内容，请使用“其它”菜单中的“抹掉磁盘内容…”命令。

    请帮我抹掉磁盘内容。

    注意：在您抹掉磁盘内容时，“格式”弹出式菜单会提供包括“Mac OS 标准格式”（HFS）和“Mac OS 扩充格式”（HFS +）的选项。“Mac OS 标准格式”与所有的 “Mac OS 格式”兼容。但“ Mac OS 扩充格式”只与使用 Mac OS 8.1 或以后版本的计算机兼容。如果用“ Mac OS 扩充格式”格式化磁盘 ，使用 Mac OS 8 或以前版本的计算机不能使用该磁盘。

    如果您不能抹掉硬盘内容：
    • 确保磁盘不是当前启动磁盘。
    • 请确保磁盘没有在网络上共享。要关闭文件共享，请使用“文件共享”控制板。
    请为我关闭文件共享。
    请为我打开文件共享。
