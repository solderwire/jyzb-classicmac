将 PowerBook 用作外接硬盘

    有些型号的 PowerBook 可以与其它计算机连接，并用作外接硬盘。只在配有外置 SCSI 连接器的 PowerBook 上才提供这项称为“SCSI 磁盘模式”的功能。

    在将 PowerBook 用作外接硬盘之前，您需要设置它的 SCSI ID，并确信已关闭密码保护功能。

设置 PowerBook 的 SCSI ID

    要设置 SCSI ID，请使用“PowerBook SCSI 设置”控制板。

    重要事项：若您将 PowerBook 连接到现有的 SCSI 链上，应确保给它一个与其它设备不同的 ID 号。

    请为我打开“PowerBook SCSI 设置”控制板。

关闭密码保护功能

    要关闭密码保护功能，请使用“密码保护”控制板。

    请为我打开“密码保护”控制板。

将 PowerBook 连接至另一台计算机

    要将 PowerBook 连接至另一台计算机，您需要使用 SCSI 磁盘适配器（另一台计算机必须有一个 SCSI 端口或装有 SCSI 卡）。要获得更多帮助，请参阅随 PowerBook 提供的使用手册。

    重要事项：若 PowerBook 硬盘以“Mac OS 扩充格式”（HFS +）格式化，则您要连接的计算机必须使用 Mac OS 8.1 或以后版本的操作系统。
