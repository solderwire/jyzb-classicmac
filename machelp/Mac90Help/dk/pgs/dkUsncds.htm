使用 CD 与 DVD 光盘

使用 CD-ROM 与 DVD-ROM 光盘

    您可以象使用其它文件一样使用 CD-ROM 光盘上的应用程序和文件。

    您不能在 CD-ROM 或 DVD-ROM 光盘上写入任何内容。若想更改光盘上的文件，您必须先将它们拷贝至其它类型的光盘上。

    有关连接 CD-ROM 或 DVD-ROM 驱动器、安装其运行所需的软件以及插入光盘的信息，请参阅随驱动器提供的说明文件。

    您可以使用 DVD-ROM 驱动器播放 DVD-ROM 和 CD-ROM 光盘，但不可以使用 CD-ROM 驱动器播放 DVD-ROM 光盘。

    注意：要观看 DVD 视频光盘，您需要附加的硬件和软件。要获得更多的信息，请向苹果授权的经销商咨询。

使用 DVD-RAM 光盘

    如果您有以下配置，则可以使用 DVD-RAM 光盘
    • 一个 DVD-RAM 驱动器或一个较新的（第 3 代）DVD-ROM 驱动器
    • 将“苹果 CD/DVD 驱动程序”和“UDF 宗卷取用”功能扩展安装到“功能扩展”文件夹中（在“系统文件夹”中）

    使用 DVD-RAM 光盘存储信息，与在任何硬盘、软盘或其它可移动可记录介质上一样。

    您可以用“通用磁盘格式”（UDF）、“Mac OS 标准格式”、“Mac OS 扩充格式”或“DOS 格式”格式化 DVD-RAM 光盘。UDF 格式允许您与支持 UDF 1.5 的任何计算机平台共享信息。

    如果您将 DVD-RAM 光盘用作启动磁盘，请以“Mac OS 标准格式”或“Mac OS 扩充格式”将其格式化。
