初始化磁盘

    使用新磁盘之前，您需要将其初始化（或格式化）。计算机内部的硬盘已在出厂时进行了初始化，因此只在硬盘损坏并且无法使用“磁盘修理工具”（或其它磁盘修理程序）进行修理的情况下，才需要进行初始化。

    警告：初始化磁盘会抹掉其中的所有信息。初始化磁盘之前，尽可能地恢复所有信息并将其拷贝至另一个磁盘。

初始化 Zip 或 Jaz 磁盘

    要初始化未格式化的 Zip 或 Jaz 磁盘，请使用 Iomega“Tools”程序 （位于硬盘的“Apple Extras”文件夹中）并根据您使用的驱动器类型（例如 USB 或 SCSI）选取适当的实用程序。

    请为我打开“Tools”。

    如果不能打开 Iomega“Tools”程序：
    确保 Zip 或 Jaz 磁盘驱动器已经连接到您的计算机上。如果磁盘驱动器没有连接到计算机上，“Tools”程序可能显示一个错误信息，声明您安装了错误版本的驱动程序。

初始化 DVD-RAM 光盘

    要初始化 DVD-RAM 光盘，将光盘插入 DVD-RAM 驱动器，并采取以下步骤之一：
    • 如果您打算将该光盘用作启动磁盘，请用“磁盘设置”应用程序以“Mac OS 标准格式”或“Mac OS 扩充格式”初始化该光盘。
    请为我打开“磁盘设置”。
    注意：您可以使用“磁盘设置”以其它格式初始化光盘，但只有以“Mac OS 标准格式”或“Mac OS 扩充格式”初始化的光盘可以用作启动磁盘。
    • 如果您不打算将该光盘用作启动磁盘，可以这样初始化它：打开“其它”菜单，选取“抹掉磁盘内容…”并在“格式”弹出式菜单中为光盘选定格式。

    注意：如果您使用“磁盘设置”应用程序并选定以“Mac OS 标准格式”或“Mac OS 扩充格式”初始化 DVD-RAM 光盘，可以将它用作启动磁盘。您不能使用以其它方法或 UDF 格式初始化的 DVD-RAM 光盘来启动计算机。

初始化硬盘、盒式磁带或光盘

    要初始化硬盘、盒式磁带或光盘，请打开“磁盘设置”（位于硬盘的“实用程序”文件夹中）。

    请为我打开“磁盘设置”。

    注意：在您初始化磁盘时，“格式”弹出式菜单会提供包括“Mac OS 标准格式”（HFS）和 “Mac OS 扩充格式”（HFS+）的选项。“Mac OS 标准格式”与所有 Mac OS 版本兼容；“Mac OS 扩充”格式只与使用 Mac OS 8.1 或以后版本的计算机兼容。如果用“Mac OS 扩充格式”格式化硬盘，使用较早版本的 Mac OS 的计算机不能使用该磁盘。

    如果硬盘不是苹果硬盘：
    请使用磁盘制造商提供的（而非“磁盘设置”）实用程序软件。

    如果您要初始化启动磁盘：
    请从系统软件光盘上启动计算机。打开“磁盘设置”，选择启动磁盘，再点按“初始化…”按钮。

    如果初始化失败：
    请打开“功能”菜单，选择“初始化选项…”，选择“低级格式化”和“将所有数据清零”选项，然后再次尝试初始化磁盘。

    如果初始化再次失败：
    请测试硬盘是否出现物理损坏。
