更有效率的檔案夾結構

    Mac OS 9.2 的檔案夾結構讓您在尋找檔案或應用程式時更為簡易方便。以下是一些您將發現到的變更：
    • “應用程式”檔案夾更名為 Applications (Mac OS 9)。
    • Apple Extras 和“工具程式”檔案夾移至 Applications (Mac OS 9) 檔案夾中。
    • Netscape Communicator 和 Internet Explorer 移至 Applications (Mac OS 9) 檔案夾中。

    請瀏覽您的硬碟以熟悉新的 Mac OS 系統，並使用 Sherlock 來尋找所有的檔案和檔案夾。
