清除磁碟內容

    清除磁碟、重新格式化或初始化磁碟，都會永久刪除磁碟上的所有內容。清除磁碟之前，請先備份所有要保留的檔案。

    若要清除磁碟或可抹寫式光碟，請使用“特別功能”清單中的“清除磁碟內容…”指令。

    請幫我清除磁碟內容。

    注意：在您清除磁碟內容時，“格式”彈出式清單會提供包括“Mac OS 標準格式（HFS）”和“Mac OS 擴充格式（HFS Plus）”的選項。“Mac OS 標準格式”與所有版本的 Mac OS 相容。但“Mac OS 擴充格式”只與 Mac OS 8.1 或以上的版本相容。使用 Mac OS 8 或較早版本的電腦不能使用“Mac OS 擴充格式”格式化的磁碟。

    若需要關於可用格式的資訊，請打開“輔助說明”清單，選擇“顯示旁白”，然後點選想進一步瞭解的選項。

    如果您不能清除硬碟內容：
    • 確定磁碟不是現用的啟動磁碟。
    • 請確定磁碟沒有在網路上共享。若要關閉檔案共享，請使用“檔案共享”控制面板。
    請替我關閉檔案共享。
    請替我打開檔案共享。
