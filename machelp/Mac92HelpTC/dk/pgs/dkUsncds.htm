使用光碟和 DVD 光碟

使用 CD-ROM 和 DVD-ROM 光碟

    您可以像使用其他檔案一般，來使用 CD-ROM 光碟上的應用程式與檔案。

    您不能在 CD-ROM 或 DVD-ROM 光碟上寫入（儲存）任何內容。若想更改光碟上的檔案，您必須先將檔案拷貝至其他類型的光碟上。

    您可以在 DVD-ROM 光碟機上使用 DVD-ROM 與 CD-ROM 光碟，不過不能在 CD-ROM 光碟機上使用 DVD-ROM 光碟。

使用 DVD-RAM 光碟

    您可以使用 DVD-RAM 光碟儲存資訊，方法就和在硬碟、磁片或其他抽取式的可燒錄的媒體上儲存資訊一樣。

    您可以使用“通用磁碟格式”（UDF，Universal Disk Format）、“Mac OS 標準格式”或“Mac OS 擴充格式”格式化 DVD-RAM 光碟。UDF 格式能讓您與所有支援 UDF 1.5 的電腦平台共享資訊。

 使用可燒錄的光碟

    若您的 Mac 配有蘋果內置光碟燒錄機，您可以製作個人專屬的光碟。

    一旦 CD-R 完成燒錄，所有檔案都會“燒錄”至 CD-R 中，且這些檔案都不能再清除或修改。

    CD-RW（可讀╱寫）光碟的內容可以清除，並能重複使用。許多舊型的電腦無法讀取這種類型的光碟，而且這種光碟與多數的音樂光碟播放器不相容。此外，CD-RW 光碟也會磨損。若希望能長期保存，且內容品質不受影響，請使用 CD-R 光碟。
