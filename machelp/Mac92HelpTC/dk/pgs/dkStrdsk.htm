從不同的磁碟開機

    若您有一個以上的硬碟或想要從系統軟體 CD-ROM 光碟或外接硬碟開機，您可以選擇磁碟讓電腦作為啟動磁碟。

    秘訣： 在某些較新的 Macintosh 電腦上，在電腦開機時，您可以按住 Option 鍵然後按一下想要作為啟動磁碟的磁碟圖像，來選擇啟動磁碟。

    若要選擇啟動磁碟，請使用“啟動磁碟”控制面板。

    請替我打開“啟動磁碟”控制面板

    重要事項： 若選了無效的啟動的磁碟（您的電腦無法從此磁碟開機），您的電腦可能會花上兩分鐘長的時間來找出有效的啟動磁碟，然後適當的開機。即使圖像出現在控制面板中，RAM 磁碟仍然無法作為啟動磁碟。若您選取了 Network Disk 圖像，而並未連接至 NetBoot 伺服器，電腦可以會花上相當長的時間開機。

從 NetBoot 伺服器開機

    若您想要從 NetBoot 伺服器開機，請在“啟動磁碟”控制面板中選取 Network Disk 。這將會增加電腦開機所需的時間。

    注意： Network Disk 圖像在電腦上一直是有效的，可以用來從 Network Disk 開機。但是，只有在網路上有 NetBook 伺服器時，才能從 Network Disk 開機。

何謂啟動磁碟？

    啟動磁碟即是內含帶有系統軟體的“系統檔案夾”。（電腦需要系統軟體來作業。）每個啟動磁碟應該都只有一個“系統檔案夾”。

    啟動磁碟的圖像通常位於桌面的右上方。若您不知道哪一個磁碟是啟動磁碟，您可以打開“啟動磁碟”控制面板來查看，現用的啟動磁碟是反白的。

    注意： 若您從光碟或 DVD 光碟開機，光碟的圖像會出現在“啟動磁碟”控制面板中，但圖像並沒有反白。
