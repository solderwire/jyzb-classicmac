初始化磁碟

    使用新磁碟之前，您需要對其進行初始化（或格式化）。電腦內部的硬碟已在出廠時進行了初始化，因此只有在硬碟損壞並且無法使用“磁碟修理工具”（或其他磁碟修理程式）進行修理的情況下，才需要進行初始化。

    警告：初始化磁碟將會清除其內所有的資訊。初始化磁碟之前，您可以將想要保留的檔案備份到另一個磁碟上。

初始化 Zip 或 Jaz 磁碟

    若要初始化 Zip 或 Jaz 磁碟，請使用 Iomega Tools 程式（位於硬碟裡的 Apple Extras 檔案夾中），並選取適用於磁碟機種類的工具程式，例如 USB 或 SCSI。

    請替我打開 Tools。

    如果 Iomega Tools 程式無法打開：
    請確定電腦已經連接 Zip 或 Jaz 磁碟機。如果未連接磁碟機，Tools 程式可能會出現錯誤訊息，告訴您安裝了錯誤版本的驅動程式。

初始化 DVD-RAM 光碟

    若要初始化 DVD-RAM 光碟，請依照下列步驟：
    1 將光碟插入 DVD-RAM 光碟機。
    2 打開“特別功能”清單，選擇“清除磁碟內容…”。
    3 在“格式”彈出式清單裡選擇磁碟的格式。

初始化硬碟、匣式磁帶或光學式磁片

    若要初始化硬碟、匣式磁帶或光學式磁片（不是光碟），請使用“磁碟設置”（位於硬碟中的“工具程式”檔案夾裡）。

    請替我打開“磁碟設置”。

    注意：當您在初始化磁碟時，“格式”彈出式清單會提供包括“Mac OS 標準格式（HFS）”和“Mac OS 擴充格式（HFS Plus）”的選項。“Mac OS 標準格式”與所有版本的 Mac OS 相容。但“Mac OS 擴充格式”只與使用 Mac OS 8.1 或以上版本的電腦相容。使用較舊版本 Mac OS 的電腦不能使用“Mac OS 擴充格式”格式化的磁碟。

    如果硬碟不是蘋果出品的硬碟：
    請使用磁碟製造商提供的磁碟工具程式，而非“磁碟設置”。

    如果您要初始化啟動磁碟：
    請使用系統軟體光碟啟動電腦。打開“磁碟設置”，選擇啟動磁碟，再按“初始化…”一下。

    如果初始化失敗：
    請打開“功能”清單，選取“初始化選項…”。請勾選“低階格式化”和“將所有資料變零”這兩個註記框。然後再嘗試初始化磁碟。

    如果初始化再次失敗：
    請測試硬碟是否出現實體損壞。

初始化可抹寫式光碟

    在您插入空白 CD-RW 或 CD-R 光碟時，螢幕上會出現訊息，請您初始化光碟。

    若 CD-RW 光碟中已存有檔案，您可以初始化光碟，這樣就可以再度使用這張光碟。這個動作將會清除光碟內所有的檔案。請查看光碟規格，以取得關於光碟使用期限的資訊。如果清除次數頻繁，光碟內容可能會變得不正確。

    您不能清除 CD-R 光碟。一旦將檔案寫入 CD-R，這些檔案就不能再更改或刪除。

    若要初始化 CD-RW 光碟，請依照以下步驟進行：
    1 將光碟插入光碟機。
    2 打開“特別功能”清單，選擇“清除磁碟內容…”。
    3 在“格式”彈出式清單裡選擇磁碟的格式。

    請參閱製作光碟，以取得更多資訊。
