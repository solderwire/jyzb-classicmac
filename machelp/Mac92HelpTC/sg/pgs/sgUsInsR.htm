安裝與刪除語言

    您可以使用“Mac OS 安裝程式”（系統軟體光碟上）來安裝及刪除不同語言的軟體。若要加入及刪除語言，請進行自定安裝，然後選取“語言套件”選項。

    注意：使用“Mac OS 安裝程式”刪除語言不會刪除兩種以上語言所使用的檔案，也不會刪除不是使用“安裝程式”所安裝的字體。

    如果您要安裝阿拉伯語和波斯語這兩種語言軟體：
    您無法同時安裝阿拉伯語和波斯語這兩種語言軟體。若要使用波斯語，您必須先刪除阿拉伯語軟體（如果已安裝此軟體）。若要安裝波斯語此語言軟體，請參閱您系統軟體光碟上之“光碟附加程式”檔案夾內的 Language Kits CD Extras 檔案夾中的“請先閱讀”。

    如果您在使用與 WorldScript 不相容的應用程式時發生問題：
    暫時停用輔助語系，方法為在電腦重新開機時，按住 Option 鍵和空白鍵。

    如果檔案或檔案夾名稱未適當顯示：
    打開“外觀”控制面板、按“字體”標籤一下，然後選擇一個適於正在使用之語言的字體。例如，如果是日文的檔案或檔案夾名稱，請選擇 Osaka 字體。

    如果應用程式清單和對話框訊息未適當顯示：
    拖應用程式到 Language Register 程式（在“工具程式”檔案夾內），然後指出其應使用何種語言才能顯示其清單和對話框。若要重新更改該應用程式的語言，請打開“語言註冊”，然後選擇新的語言。您也可以對不同的應用程式和語言製作多重“語言註冊”。
