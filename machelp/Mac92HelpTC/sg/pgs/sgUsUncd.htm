使用 Unicode 檢視多種語言的文字

    Mac OS 9.2 支援 Unicode，Unicode 能讓您製作並打開含有特殊字元、多種語言文字以及數學符號的文件。

    您可以藉由 Mac OS 檢視現有的 Unicode 文件，但若要輸入 Unicode 字元，您必須先安裝系統軟體光碟中的軟體。

    若要打開 Unicode 輸入法：
    1 請打開系統軟體光碟中，“光碟附加程式”裡的 Unicode 檔案夾。
    2 將位於“將所含項目拖到系統檔案夾中”的 Unicode 檔案及四個鍵盤佈局拖到“系統檔案夾”中（無須將其開啟）。若要打開“關於 Unicode”檔案，請將 WorldText（在系統軟體光碟上“光碟附加程式”檔案夾中的 WorldText 檔案夾裡）拖到硬碟中。
    3 將“模擬鍵盤”9.0 拖到硬碟中。
    新版本的“模擬鍵盤”能讓您輸入 Unicode，並觀看僅使用 Unicode 鍵盤的佈局。您可將新舊“模擬鍵盤”放在同一個檔案夾中，但您必須更改其中一者的名稱。
    4 重新啟動電腦。

    帶有旗幟圖像的清單中包含了您安裝的所有鍵盤。 Unicode 鍵盤和其他鍵盤中的特殊 Unicode 字元只供能辨別 Unicode 的應用程式（例如 WorldText）使用。
