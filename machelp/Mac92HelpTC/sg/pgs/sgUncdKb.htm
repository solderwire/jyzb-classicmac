Unicode 鍵盤佈局

    若要使用 Unicode 檔案夾中的鍵盤佈局（在系統軟體光碟中，“光碟附加程式”檔案夾的 Unicode 檔案夾裡），您必須啟用 Unicode 語系，並使用可顯示 Unicode 字元的應用程式（例如 WorldText）。

      鍵盤佈局名稱   您可以輸入    
    衍生的羅馬符號 包含羅馬字母語系中各種帶有重音符號的字元
    Symbol Symbol 字體中的特殊字元
    Dingbats Zapf Dingbats 字體中的特殊字元
    Unicode 十六進位輸入 所有您知道屬於四位數十六進位碼的 Unicode 字元。您可在 Unicode 組織的網站（網址：http://www.unicode.org/）找到這些碼。
