以 Mac OS X 開機

    若您的電腦隨機安裝有 Mac OS X，或您自行安裝了 Mac OS X，您可以切換使用 Mac OS 9 或 Mac OS X。

    若要設定電腦使用 Mac OS X，請使用“啟動磁碟”控制面板。

    請替我打開“啟動磁碟”控制面板

    1 按一下內含有作業系統檔案夾的硬碟旁的三角形。
    2 選取 Mac OS X 系統作為您的啟動磁碟。
    3 按“重新開機”一下。

    一旦您設定電腦使用 Mac OS X，電腦將會一直使用 Mac OS X 開機（除非您又設定為使用 Mac OS 9）。
