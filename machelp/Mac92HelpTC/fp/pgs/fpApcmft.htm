打開 DOS 或 Windows 檔案時的問題

    電腦可以辨識大部份常用的 DOS 和 Windows 檔案，並使用事先指定的應用程式來打開這些檔案。

    如果連按二下圖像也無法打開 DOS 或 Windows 的檔案：
    請嘗試在您要用來檢視或編輯該檔案的程式裡將檔案打開。 許多的 Mac OS 程式可以直接打開 DOS 和 Windows 檔案。

    若 DOS 或 Windows 檔案顯示不正確，或是出現奇怪的字元：
    • 應用程式可能會要求您依照特別的指示使用不同的檔案格式來打開和儲存文件。請參閱程式所附的文件。
    • 請嘗試使用其他的程式來打開文件。
    • 部份可以在 DOS 電腦裡正常顯示的字元無法準確的在 Macintosh 電腦上顯示。

    如果事先指定的應用程式無法正確的運作：
    您可以使用“檔案轉換程式”控制面板裡的 PC Exchange 部份來更改指定的應用程式或加入新的應用程式。

    請幫我指定應用程式為 DOS 或 Windows 檔案類型。

    您在“檔案轉換程式”控制面板裡設定的選項會修改您已經在 Internet 控制面板裡所做的轉譯設定。在 Internet 控制面板裡所做的更改亦會修改“檔案轉換程式”控制面板裡的設定。

    秘訣：若要取得更多使用“檔案轉換程式”控制面板的輔助說明，請打開“輔助說明”清單並選取“顯示旁白”。然後將滑鼠指向您有興趣的功能或清單項目。
