對於密集使用磁碟之程式增進其執行效率

    您可以使用 RAM 磁碟來加速經常要讀寫磁碟的程式之執行效率。RAM 磁碟可讓您使用電腦中部份的內建隨機存取記憶體（RAM）作為高速儲存磁碟。

    注意：使用 RAM 磁碟也能節約筆記型電腦的電池能量。

製作 RAM 磁碟

    若要製作 RAM 磁碟：
    • 在“記憶體”控制面板中打開 RAM 磁碟選項，調整 RAM 磁碟大小，然後重新啟動電腦。
    請替我打開“記憶體”控制面板。

    在某些電腦上，例如 PowerBook，您可以在“記憶體”控制面板中選擇“關機時儲存”選項來儲存 RAM 磁碟上的內容。

    警告：如果沒有在“記憶體”控制面板中選擇“關機時儲存”，關閉電腦時，電腦 RAM 磁碟上的內容將被清除。請確定在關閉電腦之前已把 RAM 磁碟上的內容拷貝到了硬碟上。

清除 RAM 磁碟的內容

    若要清除 RAM 磁碟：
    1 連按兩下 RAM 磁碟圖像把它打開。
    2 把您想儲存的檔案拷貝到硬碟或磁片上。
    3 把 RAM 磁碟上的所有東西拖到“垃圾桶”裡。
    4 打開“特別功能”清單，選擇“清除垃圾…”。

    注意：消除 RAM 磁碟內容，並不能刪除它的圖像或增加記憶體。要釋出 RAM 磁碟使用的記憶體，您必須刪除 RAM 磁碟並重新開機。

重新配置 RAM 磁碟的大小

    若要重新配置 RAM 磁碟的大小：
    • 清除 RAM 磁碟的內容，然後在“記憶體”控制面板調整 RAM 磁碟的大小，並重新開機。

刪除 RAM 磁碟

    若要刪除 RAM 磁碟：
    • 清除 RAM 磁碟的內容，然後在“記憶體”控制面板中關閉 RAM 磁碟並重新開機。
