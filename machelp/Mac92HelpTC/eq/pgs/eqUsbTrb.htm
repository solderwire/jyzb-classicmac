使用 USB 時的問題

    如果 USB 設備無法使用：
    • 若螢幕上的訊息顯示您缺少操作設備必備的軟體，您可以連接 Internet 並自動下載使用設備所需的軟體。您也可以聯絡設備製造商詢問如何取得並安裝必需的軟體。
    • 若該設備有電源線，請確定電源線已插入可用的插座，並且已打開設備的電源。
    部份的 USB 設備，例如印表機，開機後須要一些時間來暖機。如需相關資訊，請參閱設備隨附的文件。
    • 將設備的接頭從 USB 埠上拔下，稍候幾秒鐘，然後再重新插上。
    • 請確定 USB 設備接線的接頭已經正確的插上，且接線未受磨損。
    若已經磨損，請更換新的接線或聯絡設備或接線的製造商。

    如果 USB 設備是連接至集線器上（或是做為集線器的設備上）：
    • 如果集線器有電源線，請確定電源線已經插在可用的插座上。
    • 請確定集線器的 USB 接線已正確的連接至電腦。
    • 請嘗試將設備插在不同的集線器埠上，或是不同的集線器上。
    • 請嘗試將設備直接插上電腦。
