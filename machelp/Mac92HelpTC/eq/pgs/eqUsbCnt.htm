連接 USB 設備至您的電腦

    您可以在電腦開啟時連接 USB 設備至電腦上。

    警告：當正在使用 USB 設備時，請不要插上或拔掉 USB 設備（特別是儲存設備），否則將可能損壞檔案。

    若要連接 USB 設備至您的電腦：
    1 如有需要，請將 USB 設備連接電源。
    2 若您的 USB 設備沒有 USB 接線，請將 USB 接線上的 B 類型 USB 接頭插入設備所對應的 USB 埠。
    3 將 USB 接線的另一端插入電腦上可用的 USB 埠。（請參閱電腦隨附的書面文件來找出 USB 埠的位置。）

    如果所要連接的設備沒有 USB 埠：
    • 您可以使用 USB 轉換器連接非 USB 設備。

    若電腦沒有可使用的 USB 埠：
    • 增加一個 USB 集線器並將其他的 USB 設備連接到集線器的埠上。
    • 拔掉未使用的設備來增加可用的埠。

    如果 USB 設備無法使用：
    • 請參閱 USB 問題處理。
