USB 埠、接頭及接線

    USB 埠和接頭的類型為 A 類型或 B 類型。標準的 USB 接線一端是 A 類型接頭，另一端則是 B 類型接頭。為求最佳的執行效率，請使用標準的接線。

    A 類型的埠和接頭的形狀為長方形，用以連接設備至電腦或集線器上的 A 類型 USB 埠。

    A 類型的 USB 埠和接頭

    B 類型埠和接頭為正方形並用以連接 USB 接線至 USB 設備上。

    B 類型的 USB 埠和接頭

    部份設備，例如蘋果 USB 鍵盤，有一條已連接的（或稱“固定的”）接線，所以只有一端是可任意使用的 A 類型接頭。

    許多設備都隨附標準的 USB 接線。若您須要購買 USB 接線，請至電器用品店或蘋果授權的經銷商。
