USB 集線器的種類

    USB 集線器是用以提供其他 USB 設備插入插頭使用的設備。當您要同時連接更多的設備至電腦上，或是要增加設備和電腦之間的距離時，您便可以使用 USB 集線器。您可以同時插入數個集線器來擴充可用的 USB 埠，並構成大型的 USB 設備階層。您也可以使用 USB 集線器來增長相連設備之間的距離。

    有兩種類型的 USB 集線器：自行供電的集線器和匯流排供電的集線器。

自行供電的集線器

    此類的集線器有 AC 轉換器可由外部提供電源。電腦本身就是一個自行供電的集線器。若要使用集線器，您必須將轉換器的接線插入插座或延長線。如此便能提供集線器的電源，讓您連接大多數的 USB 設備。 一般自行供電的集線器有四個或七個 USB 埠。

    注意：為求最佳的執行效率，蘋果電腦建議您使用自行供電的集線器取代匯流排供電的集線器。

匯流排供電的集線器

    此類集線器的電源來自電腦或其他自行供電的集線器，所以您毋須將其插入 AC 電源插座。您可以將低用電量的 USB 設備，例如滑鼠或搖桿，插入匯流排供電的 USB 集線器。蘋果 USB 鍵盤內含一個匯流排供電的集線器。

    匯流排供電的集線器

    重要事項：不可將兩個匯流排供電的集線器連接在一起，因為這樣沒有足夠的電力可供兩個集線器及其所連接的設備使用。若要將兩個集線器連接在一起，其中至少要有一個是自行供電的集線器。因此不可將蘋果 USB 鍵盤連接至另一個蘋果 USB 鍵盤上的 USB 埠。若要在電腦上使用兩個匯流排供電的集線器，請分別將其連接至電腦的 USB 埠，或自行供電的集線器上。如果您使用三個或以上的集線器相連在一起，請在此串列中交互使用兩種不同供電型式的集線器。
