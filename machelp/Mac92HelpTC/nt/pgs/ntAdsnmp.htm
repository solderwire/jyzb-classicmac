使用 SNMP 來管理網路

    您可以使用系統軟體光碟提供的 SNMP Administrator 程式，在使用“簡易網路管理通訊協定（SNMP）”的網路上設定並監控電腦群組。

    請替我打開 SNMP Administrator。

    如果 SNMP Administrator 不在硬碟上：
    未安裝 SNMP Administrator 程式。若要安裝 SNMP Administrator，請使用電腦隨附軟體安裝光碟上的軟體。（此程式置於“光碟附加程式”檔案夾內 Network Extras 檔案夾中的 Open Transport 檔案夾裡。）

相關資訊

    若需 SNMP 網路管理的相關資訊，請查看 SNMP Administrator 應用程式的電子手冊（SNMP Admin Manual.pdf），其置於 Mac OS 光碟或電腦隨附軟體安裝光碟上的 Network Extras 檔案夾裡。
