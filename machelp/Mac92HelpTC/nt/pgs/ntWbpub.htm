發行網頁

    您可以從您的硬碟上的檔案夾使用“個人網頁共享”在 Internet 上（或是公司、學校的 Intranet）發行網頁或共享檔案。

    若要發行網頁：
    1 將 HTML 格式的檔案拷貝至硬碟上的“網頁”檔案夾。
    2 如有需要，連接至 Internet。
    3 使用“網頁共享”控制面板來指定做為主頁的檔案。主頁是訪客造訪您網站時的第一頁畫面。
    請替我打開“網頁共享”控制面板。
    4 使用“網頁共享”控制面板來啟動“網頁共享”。

    重要事項：置於“網頁”檔案夾中的檔案必須為 HTML 格式，可讓使用者以網頁方式進行檢視。

    若要共享任何類型的檔案：
    • 在“網頁共享”控制面板中選取“無”做為主頁。您的檔案即會列於使用“個人 NetFinder”的另一使用者瀏覽器視窗中。
    使用者可藉由拖檔案名稱到桌面或其他檔案夾來拷貝檔案。

    若要顯示 HTML 網頁並允許其他人拷貝列表中的檔案，請設定“個人 NetFinder”連結：
    • 在“網頁共享”控制面板中指定主頁，然後將連結加入其中一個在下列格式裡含有“PNF:”的 HTML 網頁：
    http://伺服器名稱或 IP 位址/PNF:
    http://伺服器名稱或 IP 位址/PNF:子檔案夾/

    若要檢視“個人網頁共享”的完整使用指南：
    • 使用網頁瀏覽器來檢視硬碟上“網頁”檔案夾中的“Using Personal WebSharing.html”檔案 。
