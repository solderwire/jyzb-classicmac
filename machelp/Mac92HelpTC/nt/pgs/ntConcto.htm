選擇網路服務

    某些應用程式能讓您打開指定的 URL，或瀏覽應用程式指定的服務或一般服務（如 AppleShare、FTP 或 HTTP）。

    若要瀏覽服務：
    • 看到欲瀏覽的服務時，展開所列出的網路鄰居，按一下將其選取，然後按“選取”一下。

    若要選取尋找中的伺服器類型：
    • 按“快速鍵”按鈕一下，然後選取服務類型。
     

    如果未列出尋找中的服務：
    • 選擇已選取的服務類型（例如：網頁伺服器或 FTP 伺服器）。
    • 打開 Internet 控制面板，按“進階”標籤一下，然後檢查“主機”設定中所指定的 LDAP（Lightweight Directory Access Protocol）伺服器。
    • 詢問網路管理者此服務是否已註冊於電腦可發現的網路區段中的目錄下。
    • 請參閱 Network Services Location Manager Network Administrator's Guide，以取得應用程式尋找網路服務的方法，及使用者或網路管理者所能執行的動作，使得特定的服務可列於服務列表中。您可以使用搜尋標題的方式，在下述的網站尋找此使用指南：http://www.info.apple.com/manuals/manuals.taf。

    秘訣：若要檢視附加於伺服器列表中網路鄰居裡的來源目錄類型，請在打開“檔案”對話框時按住 Control 鍵。
