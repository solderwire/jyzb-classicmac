詞彙表 — Internet 和網路

    取用權限：Mac OS 檔案共享的功能。取用權限分配給單一使用者、群組、訪客，並決定那些使用者能否看見或更改共享磁碟上的檔案。取用權限在共享項目的“簡介”視窗的“共享”部份設定。

    動作：“個人網頁共享”的功能，可讓“網頁”檔案夾啟動“通用閘道介面（CGI）”工序指令、“非同步 CGI”或其他應用程式。

    任何使用者：在 Mac OS 檔案共享中，“持有人”設定可讓每個人連接到伺服器，以進行共享項目的更改。在項目的“簡介”視窗的“共享”部份中選擇“任何使用者”後，任何使用者皆可更改共享項目的取用權限。

    蘋果遠端連線（ARA）：“蘋果遠端連線（ARA）用戶端”軟體可讓您連接到“ARA 多埠伺服器”、已安裝的“ARA 個人伺服器”軟體的電腦，或者 ARA 相容伺服器。

    AppleShare：可讓您連接到網路上其他電腦和使用共享檔案夾或磁碟內容的軟體。您也可以使用 AppleShare 來共享電腦上的檔案夾或磁碟。

    AppleTalk：一些網路設備（如：印表機和伺服器）所使用的網路通訊協定。

    瀏覽器：可讓訪客看到網頁伺服器內容的應用程式。

    CGI（通用閘道介面）工序指令：伺服器在指定時間執行的應用程式。例如，當訪客連接到伺服器，伺服器會打開 CGI 工序指令，顯示供訪客填寫的表格，然後把訪客資料發送到儲存區。

    設定：針對特定網路組件（如：AppleTalk 或 TCP/IP）的一組設定。設定是以組合的方式集結，因此您可以轉換設定組合，來轉換此組合所屬的所有設定。

    網域類型：網站名稱的字尾可以用來辨識網站所含內容的類型，或者網站所屬相關機構（例如：用於商業方面的“com”、用於教育部門的“edu”、用於政府機關的“gov”、用於非營利組織的“org”）。

    任何人：Mac OS 檔案共享的取用權限範疇，適用於“持有人”和“使用者／群組”除外的所有使用者（已註冊的使用者和訪客）。“任何人”的權限在項目“簡介”視窗的“共享”部份設定。

    “檔案共享”控制面板：用於打開或關閉檔案共享的控制面板。檔案共享允許網路上的註冊使用者取用共享檔案夾和檔案。

    訪客：沒有提供名稱連接到伺服器的任何人。訪客取用在“使用者與群組”標籤的“檔案共享”控制面板中設定，並與 Mac OS 檔案共享一起打開或關閉。

    主頁：訪客連接到網頁伺服器時，所看到的的第一個 HTML 文件。

    HTML（超本文標記語言）：一種整理文字檔案的方法，以便在全球資訊網上顯示內容及連接到其他檔案。

    Internet：以電腦和通訊硬體連接全球資訊網站和其他服務（如：電子郵件、討論群組及 FTP 伺服器），並互相連接的龐大國際網路。

    Internet 網路服務供應商（ISP）：提供直接 Internet 連線服務的公司。通常擁有快速的、專用的 Internet 連接服務，和提供給單一使用者透過電話線連接的多個數據機。當您向 ISP 申請帳號時，這家網路服務供應商會提供為連接 Internet 所需的資訊和基本軟體。

    Intranet（企業內部網路）：以企業網路為基礎的網頁伺服器和網站，通常位於一間公司或其他組織內。

    IP 位址：IP 為 Internet Protocol（Internet 通訊協定）的縮寫，一種指定資訊如何透過電腦網路進行傳送的方法。IP 位址用來辨識所指定的電腦，由 4 到 12 個被句號分成四部份的數字組成（例如：130.43.2.2）。

    MIME 類型：與網頁伺服器上的檔案有關的資訊。MIME 類型是用來告知伺服器和任何已連接使用者的網頁瀏覽器，此檔案的類型和使用那個應用程式來打開它。（MIME 是 Multipurpose Internet Mail Extension 的取首字母的縮寫。）

    數據機：使電腦能透過普通電話線與其他電腦進行通訊的設備。數據機可將數位資訊轉換成大多數電話線上使用的類比式訊號。

    網路：使用專門的軟體和硬體互相連接，以共享和傳送資料的電腦群組。

    持有人：在 Mac OS 檔案共享中，控制指定檔案夾的人。

    個人 NetFinder：“個人網頁共享”的功能，可將網頁檔案夾的內容以列表顯示。

    傳輸埠：表示兩個設備之間通訊用的分級數字；在通訊過程中，每個設備的分級必須一樣。例如，網頁伺服器和瀏覽器使用 HTTP 來通訊，所以兩者皆在 80 埠上執行。在“個人網頁共享”中，埠是電腦上網頁檔案夾的位址。

    註冊使用者：在 Mac OS 檔案共享中，具有與該名稱相關取用權限的使用者姓名。

    TCP/IP 網路連線：Transfer Control Protocol/Internet Protocol 的縮寫，一種指定資訊如何透過 Internet（或藉由網路連接）傳送的方式。

    URL（統一網路資源定位）：網站或其他 Internet 服務的“位址”或位置 。

    使用者及群組：在 Mac OS 檔案共享中“檔案共享”控制面板上的標籤，用於指定已註冊使用者、群組及訪客。

    網頁：在網頁伺服器上的任何 HTML 文件。

    網頁伺服器：執行透過 Internet 或 Intranet 共享 HTML 和其他檔案之軟體的電腦。

    “網頁共享”控制面板：用來設置“個人網頁共享”的軟體。

    網站：訪客能透過 URL 連接、網頁瀏覽器檢視的 HTML 及其他內容。
