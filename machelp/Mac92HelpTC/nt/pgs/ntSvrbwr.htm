使用網路瀏覽器

    您可以使用“網路瀏覽器”（“蘋果”清單中），連接至 AppleShare、FTP 或 HTTP 伺服器。

    請替我打開“網路瀏覽器”。

    若要連接伺服器：
    • 從列出的網路鄰居中找尋要連接的伺服器，然後按其名稱兩下。
    秘訣：您可以使用向前和向後箭頭來追溯先前的行徑，或是按“快速鍵”按鈕並選取“網路”回到起點（網路鄰居的上層）。

    若要使用 URL 或 IP 位址連接伺服器：
    • 按“快速鍵”按鈕一下，並選取“連接至伺服
    器…”，然後輸入伺服器的 URL 或 IP 位址，再按“連接”一下。
    “快速鍵”按鈕

    若要連接最近參訪的伺服器：
    • 按“最近使用”按鈕一下，然後從彈出式清單中選取伺服器名稱。
    “最近使用”按鈕

    若要連接喜好的伺服器或鄰近區域：
    • 按“喜好項目”按鈕一下，然後從彈出式清單中選取伺服器名稱。
    “喜好項目”按鈕

    若要將伺服器或鄰居加到喜好項目中：
    • 在喜好項目列表中選取項目名稱，然後按“喜好項目”按鈕一下，並從彈出式清單中選取“加到喜好項目”。
    您也可以將喜好項目列表中的項目拖到“喜好項目”按鈕。

    秘訣：若要製作伺服器或鄰居的替身，請將項目名稱拖到桌面上。

    如果未列出尋找中的伺服器：
    • 檢查 AppleTalk 與 TCP/IP 控制面板中的設定。
    • 檢查 Internet 控制面板中“進階”標籤上“主機”設定中指定的 LDAP（Lightweight Directory Access Protocol）伺服器。
    • 詢問網路管理者此伺服器是否已註冊於電腦可發現的網路區段中的目錄下。
    • 請參閱 Network Services Location Manager Network Administrator's Guide，以取得“網路瀏覽器”尋找服務的方法，及使用者或網路管理者所能執行的動作，使得服務可出現於瀏覽器中。您可以在 www.info.apple.com/manuals 網站中尋找此使用指南。
