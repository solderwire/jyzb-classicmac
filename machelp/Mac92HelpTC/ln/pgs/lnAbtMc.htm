關於 Mac OS 9.2
    本文將提供關於 Mac OS 9.2 的最新即時資訊。
    從 Mac OS 9.2 光碟啟動
    若您從 Mac OS 9.2 光碟開機，則部分的功能（例如在“Mac 輔助說明”中播放 QuickTime 影片）將無法使用。而當您從硬碟重新開機時，所有功能的使用都將會回復正常。
    使用 Acrobat PDF 文件
    Mac OS 9.2 中的部分文件是使用 Adobe Acrobat 所製作的（這些檔案的名稱結尾為“pdf”）。您必須使用 Adobe Acrobat Reader 閱讀這些文件。若要安裝 Adobe Acrobat Reader，可從 Mac OS 9.2 光碟上 Adobe Software 檔案夾中找到此軟體，以進行安裝。
    “磁碟設置”與“能源節約器”
    當您使用“磁碟設置”初始化磁碟或測試磁碟時，請將“能源節約器”控制面板中的滑桿拖到“永不”，以確定系統不會進入睡眠狀態。您可以在完成格式化之後再恢復成先前的設定。
    安裝後顯示器解析度的變更
    當您執行 Mac OS 9.2 的另建系統之後，顯示器解析度可能會與先前的設定有所不同。您可以使用“顯示器”控制面板，將顯示器解析度回復至原先的設定。

關於相容性的注意事項

    FontSync 設置
    FontSync 1.0 所製作的“FontSync 設置”與字體參考無法與 Mac OS 9.2 內的 FontSync 2.0 搭配使用。請使用 FontSync 2.0 製作新的“FontSync 設置”與字體參考。
    透過 IP 使用“檔案共享”
    當您透過 IP 使用“檔案共享”時遇到下述問題：
    • 在更改網路設定之後喚醒系統時發生延遲狀況
    • 在“啟動”或“關機”時發生延遲狀況
    • 在開始或停止“檔案共享”時發生延遲狀況
    請使用“延伸功能管理程式”控制面板，停用 SLPPlugin 延伸功能。
    蘋果光碟機延伸功能和非蘋果 CD/DVD 光碟機的不相容性
    若您使用的非蘋果 CD 或 DVD 光碟機發生問題，或是您所使用的 CD 或 DVD 光碟機是蘋果所製造但使用的是非蘋果製造的驅動程式，您可能要從“延伸功能”檔案夾（位於“系統檔案夾”）裡刪除“蘋果 CD/DVD 驅動程式”這個檔案，然後重新安裝 CD/DVD 光碟機隨附的軟體。請聯絡 CD/DVD 光碟機的製造商，以確定您所使用的是否為最新版本的軟體。
    安裝 QuickTime 的應用程式
    部分應用程式會以舊版本的 QuickTime 取代 Mac OS 9.2 所安裝的 QuickTime。若發生這種情形，您必須在“Mac OS 安裝”中選擇“加入／刪除”來重新安裝 QuickTime，然後執行自定安裝。如需更多的輔助說明，請打開“Mac 輔助說明”並尋找關於“增加或刪除系統軟體”的主題。
    “蘋果硬碟設置”程式
    如果在安裝 Mac OS 9.2 之後您需要初始化硬碟，請使用 Mac OS 9.2 光碟上的“磁碟設置”程式。請勿使用“蘋果硬碟設置”程式。
    FireWire 硬碟
    FireWire 2.8.2 包含了一個新的內建驅動程式，可供多種 FireWire 硬碟使用。如果找不到廠商提供的驅動程式延伸功能，便會自動使用此驅動程式。您仍然應該安裝廠商所提供的軟體，如此才能使用 FireWire 儲存設備的最多功能。

關於列印的注意事項

    在 LocalTalk 網路和序列印表機之間切換
    若您使用相同的埠來交互連接 LocalTalk 網路和序列印表機，在關閉 AppleTalk 以使用印表機時，連接埠可能未被正確的重置。如果您在關閉 AppleTalk 之後有列印上的問題，請重新開機。
    重新安裝印表機驅動程式
    若您執行 Mac OS 9.2 的另建系統安裝但無法在“選擇週邊”裡找到印表機的圖像，您可能必須重新安裝印表機隨附的軟體。

    • 若您使用的是 StyleWriter、StyleWriter II 或 StyleWriter 1200，請使用 Mac OS 9.2 隨附的 Color SW 1500 驅動程式（而不是安裝原來的印表機軟體）。

    • 若您使用的是 Color StyleWriter 2200 或 2400，請安裝 Mac OS 9.2 隨附的 Color StyleWriter 2500 驅動程式（而不是安裝原來的印表機軟體）。

    • 若您使用的是 Color StyleWriter 4000 系列印表機，您可以在系統軟體光碟的“光碟附加程式”裡找到最新版的驅動程式。

    您可能也必須重新安裝印表機軟體隨附的字體。請依照印表機軟體裡的指示進行。

關於“多重使用者”環境的注意事項

    使用 Epson 印表機
    若要在 Epson 印表機上列印，在“限制的”或“面板”環境下的使用者必須設定 Epson 印表機驅動程式，以使用位於使用者檔案夾裡的幕後列印檔案夾，或是其他能讓使用者製作檔案的位置。在 Epson 的列印對話框裡按 Tools 按鈕一下，然後再按 Configuration 一下。然後更改 Temporary Spool 和 Temporary High Speed Copies 檔案夾的位置。
    在“多重使用者”環境下取用印表機
    若印表機沒有出現在桌面上，那麼它也不會顯示在“多重使用者”控制面板的可用印表機列表裡。若要允許“限制的”或“面板”使用者使用這些印表機，請開放“所有印表機”的取用權限，或者讓使用者可以取用“選擇週邊”。
    程式相容性
    如果您在“限制的”或“面板”的使用者環境下遇到問題，請在“Mac 輔助說明”裡尋找“多重使用者”，以取得問題處理或其他關於設定軟體的重要資訊。

其他注意事項

    更改檔名將更新檔案修改日期
    如果您在 Mac OS 9.2 裡更改了檔案或檔案夾的名稱，如此也會更改包含該項目的檔案夾之修改日期。
    RAM 磁碟在重定大小時消失
    如果您在 Mac OS 9.2 系統上重設 RAM 磁碟的大小，其圖像可能會在桌面上消失。重新開機後磁碟圖像即會以新設定的大小再次出現。
    使用電源按鈕進入睡眠
    對於吸入式光碟機的 iMac 機型、G4 Cube 和大多數的 PowerMac G4 電腦，都可以按一下機身上“電源”按鈕來讓電腦進入睡眠狀態。您可以在“能源節約器”控制面板裡取消勾選“按電源按鈕讓電腦進入睡眠狀態”的註記框來關閉這項功能。
    蘋果遠端連線個人伺服器
    若您使用“蘋果遠端連線個人伺服器”並執行 Mac OS 9.2 的另建系統，則安裝完成後第一次重新開機時，伺服器可能無法正確回應撥入的電話。請在使用“蘋果遠端連線個人伺服器”之前，再重新開機一次。
