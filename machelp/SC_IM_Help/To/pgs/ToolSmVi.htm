Smart View 控制板

    Smart View 是一个控制板，随 Mac OS 8.1 或以上的中文系统安装在“控制板”文件夹内。

    在中文系统的环境下，用户安装英文应用程序时，他们的文件（在应用程序自己的文件夹或系统文件夹内）往往会带有一些特殊字符。而由于英文系统与繁体简体的基本编码不同，特殊字符在不同的码表的位置也不一样，故在中文系统下显示这些英文特殊字符时，会变成方框或黑框等乱码显示。虽然实际使用这些文件时不会有问题，但在美观方面始终是一个瑕疵。

    Smart View 便是特别为解决特殊字符的显示问题而开发的。在显示时，如果发现有这些英文字符时，Smart View 便会以一种英文字符来取代使这些特殊字符能够正确显示。

    由于 Smart View 在使用上有其他的限制（请参阅下面的说明），用户可在控制板内选择是否使用 Smart View。所作选择不需重新启动计算机便可即时生效；但正在显示的文稿则须在屏幕画面重画以后，才会显示所作的改动。

    “仅系统字体”
    由于 Smart View 在使用上有其他的限制（请参阅下节），如果用户想避免在使用其他字体时（Finder 及一般应用软件的对话框均设定使用系统字体，即 Beijing 字）也有这些限制，可选取这个选项。

    “符号显示用”
    用户可在这个弹出式菜单中选取用以显示特殊字符的字体，列出的选择通常是系统所安装的英文字体。

    注意：如果所选字体与原来所用的中文字体字宽不同的话，则会影响插入点的位置。（中文系统中建议使用 Charcoal 搭配 Beijing 使用）。

使用限制

    用户在使用 Smart View 请注意以下限制：

    所有特殊字符与中文字符或特殊字符之间必须加入一个空格键或 Return 键，才能正确显示。
