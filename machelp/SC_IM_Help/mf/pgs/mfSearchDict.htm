查询字库

    在“输入法”菜单上选取“显示字符表”，输入法便会把字符表显示出来。

    您也可利用对应的迅捷命令（在键盘上按 option-shift-B 键）来显示字符表。假如调版已显示在屏幕上，那么在调版上点按“字符表”按钮，也是显示字库的一个快捷方法。屏幕会显示如下的窗口：

    字符表窗口亦是浮动的窗口，其预设位置在屏幕中央。通过拖移其左方的标题栏，可以把窗口放置到屏幕上任何位置，移动不会影响窗口在原来位置所显示的内容。

    点按关闭格可关闭字符表窗口；拖移调整大小格则可改变窗口的大小。 若屏幕空间尺寸许可，通过拖移窗口右下角之调整大小格，可向下延伸字库窗口。要缩小窗口，只需向上拖移调整大小格。

    字库窗口上半部显示四项信息：字符表类型、设定字体、所选字和输入码。

    字符表分四种方式显示：部首笔划表、国标汉字、标点与符号和 Ｕnicode 表。字符表类型预设为部首笔划表，但您可以利用字符表类型弹出式菜单，选定所需的其它类型。

    字符表窗口预设使用系统字体（即“Beijing”字体）来显示字库，但您可以利用设定字体弹出式菜单，选用其它显示字体。指向倒三角形按钮，并按住鼠标，该菜单便会显示当前系统可用的中文字体；您可以选用菜单上的任何一种字体，作为显示字体。这也是检视每种字体字形特性的一个方法。

    当字库中的某个字被选中时，字符表类型和设定字体弹出式菜单右方会显示方格，白色方格内显示所选的字或符号的字形。该字会较下方视窗内所显示的字略大，方便检视。

    您所选取的字或符号除了放大显示在白色方格内之外，其输入码亦会随即显示在方格右方。当字符表类型设定为“国标汉字”或“部首笔划表”时，所显示的输入码依次为区位码、ABC拼音码、五笔型码和五笔画码，以方便您检视系统所有的输入码。当字符表类型设定为“Unicode 表”时，所显示的输入码依次为区位码，GB 码，Unicode 码和 GBK 码。

    字库窗口下半部列出了中文字库的内容，每行至多显示十六个汉字或符号。通过点按滚动箭头，或是拖移滚动格，用户可以检视整个字库内的每一个字和符号，以及其输入码。

    事实上，字库内容窗口亦可用来输入汉字或符号。找到需要的项目后，连按该字或符号，便可将它输入到文稿之内。这个方法虽然是较慢的输入法，但对于各种输入方法的初学者，或遇上较难组拼的字或符号，利用字库视窗来输入，不失为最保险的方法。而且可以通过窗口所显示的信息，学习该字或符号的输入方法。

部首笔划表

    当您不知道要输入的汉字的任一输入码时，可用“部首笔划表”来查找该字。用字符表类型弹出式菜单选取“部首笔划表”，部首笔划表就会显示出来。

    部首笔划表按通用汉语字典的部首索引编排方式，以汉字部首笔划为第一索引，以除部首外笔划数为第二索引来编排汉字。查询汉字时，先用部首笔划弹出式菜单选中待查汉字的部首笔划数，在选部首窗口中即会显示所有该笔划数的部首，您可从中选中待查汉字的部首。在“部首笔划表”窗口右部，排列有 “０”到“23＋”共二十四个部首外笔划数按钮，点按待查汉字除去部首后的笔划数，下方的选字窗口即会显示所有符合条件的汉字。除去部首后的笔划数为 23 画或多于 23 画的，点按“23＋”。选中待查汉字后，“部首笔划表”窗口右上部会显示该字及其输入码。您可以用输入码输入该字或连按选字窗口中的该字直接输入它。例如查询“苹”字：

Unicode 表

    Unicode 表供您查询字符的 Unicode。Unicode 是针对书写字符制定的国际统一编码。它包含世界各大语系的字符以及常用的技术符号。它以 ASCII 码为蓝本，采用十六位进行编码。Ｕnicode 表是根据苹果公司制定的 TEC 1.40 版 Ｕnicode 标准编写的。而 TEC 是用于对来自于外来系统（如 Internet 或其它非 Mac OS 系统）文件中的文字和数据进行格式转换以便于在 Mac OS 中正确显示和处理的功能扩展。
