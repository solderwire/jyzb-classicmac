编辑菜单

预置…

    用户可以使用“预置…”命令，设定程序启动时显示工具板。

    用户也可以使用“预置…”命令，设定程序启动时显示 “字符表参考” 面板，方便用户造字时选择字体样板，参考可能引用的字符的形状和查看字码表内的所有字符。

    用户可以使用 “预置…”命令，设定程序启动时打开字符编辑窗口，方便直接造字。

    假如您在预置对话框设定了程序启动时不显示工具板，但在造字时又需要使用，您可以在“窗口”菜单下选取“显示工具板”命令，或按 Command-T，便可以立刻显示工具板。

    用户可以在 TrueType 字体预置对话框设定所造字体的幅度。由于所造字符必须与其他字体和英文字符配合，故“TrueType 字体编辑程序”设计可让用户设定字体幅度。设定包括字符的“底线以上数值”、“底线以下数值”、“前字位宽度”和“全角单元”。

    首次启动“TrueType 字体编辑程序”时，程序会在“系统文件夹”之“预置”文件夹内建立一个 TrueType Font Editor Prefs 预置文件。用户每次在预置对话框更改设定，所作更改都会存储至 TrueType Font Editor Prefs 文件。
