Internet



Mac OS 8S 包括功能强大的软件，用来浏览 Web、发送电子邮件乃至发布自己的网页。

注册

“Internet 设置助理”帮助您配置系统访问 Internet。无论您采用调制解调器连接还是局域网连接，它都可帮助您创建 Internet 服务供应商（ISP）下的新帐户或是从 ISP 下的现有帐户使用设置。

连接

“Internet 拨号程序”让您通过调制解调器连接到 Internet。如果您有多个 Internet 服务供应商，“Internet 拨号程序”会允许您选择使哪个配置活跃，包括局域网配置。

浏览

使用“Internet 设置助理”设置 Internet 访问之后，点按桌面上的“浏览 Internet”按钮可轻易打开您喜欢的 Web 浏览器。

随 Mac OS 安装的有一个 Web 浏览器，Netscape Navigator。

使用电子邮件

使用“Internet 设置助理”设置 Internet 访问之后，点按桌面上的“邮件”按钮可以很容易地打开您喜欢的电子邮件应用程序。

电子邮件应用程序 Claris Emailer Lite 随 Mac OS 安装。

个人 Web 共享

“Mac OS 个人 Web 共享”让您创建自己的 Web 服务器并与 Internet 上的每个人共享文件。

在您设置服务器、往里面添加内容并且打开“Web 共享”之后，知道您服务器地址的其他人就可访问您的个人节点并浏览其内容。只要您的计算机连接到 TCP/IP 网络（Intranet 或 Internet）且您的服务器在运行，使用网络的每个用户就可得到服务器的内容，而不管他们使用的是什么浏览器或什么型号的计算机。

“连接到…”命令

使用“Internet 设置助理”设置 Internet 访问之后，您可使用“苹果”菜单中的“连接到…”命令快速连接到 Internet 上的位置（URL）。

Mac OS Runtime for Java

Mac OS Runtime for Java 让计算机运行用 Java 编写的程序，Java 是一种流行的新的程序设计语言。Java 程序或“applet” 常嵌入在 World Wide Web 上的页面中。

Mac OS Runtime for Java 也允许软件开发商在 Mac OS 上构建 Java 应用程序。

右边的 Tic Tac Toe 游戏是一个简单的 Java applet 的示例。要玩该游戏，请点按棋盘上的任何方格。（如果游戏未出现，请确保使用的是支持 Java 的浏览器，比如 Internet Explorer 3.01 或以后版本。）
