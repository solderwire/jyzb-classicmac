Internet 软件

Mac OS 包含功能强大的软件集合，可充分利用 Internet。

注册

“Internet 设置助理” 帮助您配置系统来访问 Internet。无论您是通过调制解调器或局域网连接，还是创建 Internet 服务提供商（ISP）下的新帐户，或者是从 ISP 下的现有帐户使用设置，它对您都有帮助。

连接

“Internet 拨号程序” 让您通过调制解调器连接到 Internet。如果您有多个 Internet 服务供应商，“Internet 拨号程序”会允许您选择使哪个配置成为现用，包括局域网配置。

您可使用 PPP 软件通过调制解调器连接到 Internet。

自定系统以使用 Internet

“Internet 设置实用程序” 可让您选择每次连接到 Internet 时您想打开的文稿、文件和应用程序。您也可更改当前 Internet 配置、输入或输出 Internet 配置、移走或复制配置。

Internet Config 是一种共享软件应用程序，它让您添加、修改和移走诸如电子邮件、新闻组和 World Wide Web 的 Internet 服务的配置信息。您也可为字体、文件映射和帮助程序设置预置。可阅读硬盘上的 IC_FAQ.html 文件（关于 Internet 配置经常问及的问题）来了解如何使用 Internet Config。 

浏览器和电子邮件

Internet Explorer 和Netscape Navigator 是 Internet 浏览器，帮助您浏览 World Wide Web 和访问各种 Internet 服务，例如电子邮件和新闻组。这些浏览器也允许您显示在硬盘上用超文本标记语言（HTML）创建的文稿。

Claris Emailer Lite 可让您通过 Internet 发送和接收电子邮件。

如果已连接到 Internet，可查看下列 Web 节点获取更多信息：

  • Microsoft Web 节点 网址为：(http://www.microsoft.com)
  • Netscape Web 节点 网址为：(http://www.netscape.com)
  • Claris Web 节点 网址为：(http://www.claris.com)

多媒体插件

插件可扩展 Internet 浏览器（例如 Internet Explorer）的能力，允许您显示影片、动画和虚拟环境，并允许您听语音和音乐。

  • QuickTime plug-in 可让您直接体验 Web 页中的视频、音频、动画和 VR（虚拟现实）景物。
  • LiveAudio plug-in可让您播放声音文件。
  • Shockwave plug-in 可让您体验 World Wide Web 上的交互式多媒体、图形和实时音频。

个人 Web 共享

Mac OS 个人 Web 共享 可让您创建自己的 Web 服务器并与 Internet 上的每个人共享文件

信息服务

America Online （在 Mac OS 只读光盘上）是世界上最流行的 Internet 联机服务，全世界有八百多万成员。该服务给用户提供各种交互式功能－－电子邮件、Internet 访问、新闻、体育、天气、财经信息和事务处理、电子购物，等等。如果已连接到 Internet，查看 AOL Web 节点 （http://www.aol.com）可获取更多信息。

PointCast （在 Mac OS 只读光盘上）是一种信息服务，它自动将最新信息，比如新闻头条、体育比分和股票价格下载到您的计算机上。如果已连接到 Internet，查看 PointCast Web 节点 （http://www.pointcast.com）可获取更多信息。

Marimba's Castanet Tuner （在 Mac OS 只读光盘上）可让您调谐 Internet 上的“频道”，几乎与您调谐喜欢的电视频道的方式一样。当新信息被添加到您所选节点之一时，调谐器会自动检索。如果已连接到 Internet，查看 Marimba Web 节点 （http://www.marimba.com）可获取更多信息。

辅助程序

Adobe Acrobat Reader （在 Mac OS 只读光盘上）允许您显示和打印便携文稿格式（PDF）的文件。您需要用该程序来显示 Internet 上的 PDF 文稿以及随系统软件安装的 PDF 文稿。如果已连接到 Internet，查看 Adobe Web 节点 (http://www.adobe.com) 可获取更多信息。

DropStuff 可让您快速压缩文件，方法是将文件拖到 DropStuff 图象上。压缩过的文件通过 Internet 传递效率更高。启动 DropStuff 安装器后，DropStuff 会安装在计算机上。 Stuffit Expander 允许您将从 Internet 上接收的压缩文件解压缩。如果已连接到 Internet，查看 Aladdin 系统 Web 节点 (http://www.aladdinsys.com) 可获取更多信息。
