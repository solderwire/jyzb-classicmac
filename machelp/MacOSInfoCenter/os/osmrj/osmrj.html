Mac OS Runtime for Java

用 Mac OS Runtime for Java 可做什么？

Java 是一种新的程序设计语言，它随着 Internet 的成长而得到流行 。可用Java 编写叫作“applet”的小程序并嵌入 Web 网页。当用具有 Java 功能的 Web 浏览器显示该页面时，这些程序会自动下载。

下面的 Tic Tac Toe 游戏是一个简单的 Java applet 的例子。 要玩该游戏，请任意点按一个方格。

（如果游戏未出现，请确保使用的是支持 Java 的浏览器，例如Internet Explorer 3.01 或其更新的版本。）

用 Java 编写的程序可在运行 Java 虚拟机，比如 Mac OS Runtime for Java 的任何计算机上执行。

Mac OS Runtime for Java 允许软件开发商在 Mac OS 上建立 Java 应用程序。通过使用 Mac OS Runtime for Java，开发商可建立独立的 Java 应用程序，同时可在现有应用程序内运行 Java。

随 Mac OS Runtime for Java 附有演示实用程序，这些实用程序在独立的显示程序和 OpenDoc 文稿内运行 Java applet。
