用 Mac OS 可做什么？

基础

Finder 是用来查找和组织文件并用来打开其它程序的程序。

系统文件夹 包含计算机执行基本操作所需要的软件。

Mac OS 辅助 可回答关于如何使用 Mac OS 的问题。

网络和 Internet

Open Transport 可提供连接到 AppleTalk 网络和 Internet 的软件。

苹果远程访问 可让您使用调制解调器连接到计算机网络。

Internet 软件 可让您浏览 World Wide Web 和发送电子邮件。

个人 Web 共享 可让您在 World Wide Web 上发布页面。

Mac OS Runtime for Java 可让计算机执行 Java 程序。

兼容性

PC Exchange 可让您使用 DOS 或 Windows 计算机的软盘。

MacLinkPlus 可让您容易地使用 DOS 或 Windows 计算机的文件。

多媒体

QuickTime 可让计算机显示数字化的视频。

QuickTime VR 可让计算机显示虚拟现实环境。

QuickDraw 3D 可让计算机显示三维物体。

QuickDraw GX 可让您使用高级颜色、图形和类型能力。

Text-to-Speech 软件（只在英文系统可用）可让计算机读出已写好的文本。

强大的功能

OpenDoc 可让您创建并编辑包含不同类型数据的复合文稿。

AppleScript 是可用来执行自动化任务的程序设计语言。
